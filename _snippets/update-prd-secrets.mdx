## 更新密钥

<Steps>
  <Step title="Streamlit应用密码与API密钥">
    在 `workspace/secrets/prd_app_secrets.yml` 中更新 Streamlit 应用密码和 API 密钥

    ```python workspace/secrets/prd_app_secrets.yml
    APP_PASSWORD: "admin"
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
  <Step title="RDS数据库密码">
    在 `workspace/secrets/prd_db_secrets.yml` 中更新 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # prd RDS 数据库使用的密钥
    MASTER_USERNAME: ai
    MASTER_USER_PASSWORD: "ai9999!!"
    ```

  </Step>
</Steps>
