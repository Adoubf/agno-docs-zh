| 参数 | 类型 | 默认值 | 描述 |
| --------- | ---- | ------- | ----------- |
| `collection` | `str` | `None` | 在 ChromaDB 中存储向量和元数据的集合名称 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入向量的嵌入器实例 |
| `distance` | `Distance` | `Distance.cosine` | 用于相似性搜索的距离度量方法 |
| `path` | `str` | `"tmp/chromadb"` | 使用持久化客户端时 ChromaDB 数据的存储路径 |
| `persistent_client` | `bool` | `False` | 是否使用持久化的 ChromaDB 客户端 |
| `reranker` | `Optional[Reranker]` | `None` | 可选的重排器实例，用于对搜索结果进行重排序 |
