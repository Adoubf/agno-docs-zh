| 参数        | 类型            | 默认值                      | 描述                                                                                                                       |
| ---------- | --------------- | ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `id`       | `str`           | `"deepseek-chat"`            | 用于生成响应的特定模型ID。                                                                              |
| `name`     | `str`           | `"DeepSeek"`                 | DeepSeek模型的名称标识符。                                                                                       |
| `provider` | `str`           | `"DeepSeek"`                 | 模型的提供商。                                                                                                        |
| `api_key`  | `Optional[str]` | -                            | 用于向DeepSeek服务认证请求的API密钥。从环境变量`DEEPSEEK_API_KEY`中获取。 |
| `base_url` | `str`           | `"https://api.deepseek.com"` | 向DeepSeek服务发起API请求的基础URL。                                                                     |
