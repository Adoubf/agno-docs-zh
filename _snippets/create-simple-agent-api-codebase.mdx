## 文件夹结构

`agent-api` 文件夹包含以下结构：

```bash
agent-api                     # root directory
├── agents                  # add your Agents here
├── api                     # add fastApi routes here
├── db                      # add database tables here
├── Dockerfile              # Dockerfile for the application
├── pyproject.toml          # python project definition
├── requirements.txt        # python dependencies generated by pyproject.toml
├── scripts                 # helper scripts
```

#### 预制智能体

`/agents` 文件夹包含可供直接使用的预制智能体模板。

- **网页搜索智能体**：一个能执行网络搜索的简易智能体。
- **Agno助手**：可解答关于Agno平台问题的智能体。
    - **重要提示**：使用前务必加载 `agno_assist` 知识库。
- **金融智能体**：通过Yahoo Finance API获取股票价格和金融数据的智能体。
