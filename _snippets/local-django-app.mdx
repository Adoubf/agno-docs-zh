## 本地Web应用

您的代码库已预置一个连接至[Postgres](https://www.postgresql.org/)数据库的[Django](https://www.djangoproject.com/)应用。运行以下命令启动：

<CodeGroup>

```bash 终端
ag ws up
```

```bash 完整选项
ag ws up --env dev --infra docker
```

```bash 速记
ag ws up dev:docker
```

</CodeGroup>

**按Enter键**确认并等待镜像下载完成（首次运行需时数分钟）。可通过docker仪表板查看容器状态与日志。

### 访问Django应用

打开[localhost:8000](http://localhost:8000)访问本地运行的Django应用。

![django-app-django-local](/images/django-app-django-local.png)

### Django管理后台

打开[localhost:8000/admin](http://localhost:8000/admin)访问Django管理站点。运行以下命令创建管理员用户：

```bash
docker exec -it django-dev-app python manage.py createsuperuser
```
