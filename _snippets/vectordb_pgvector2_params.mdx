| 参数         | 类型                             | 默认值           | 描述                                                                                   |
| ------------ | -------------------------------- | ----------------- | -------------------------------------------------------------------------------------- |
| `collection` | `str`                            | -                 | 用于存储向量数据的集合名称                                                            |
| `schema`     | `Optional[str]`                  | `"ai"`            | 数据库模式名称                                                                        |
| `db_url`     | `Optional[str]`                  | `None`            | 数据库连接URL                                                                         |
| `db_engine`  | `Optional[Engine]`               | `None`            | SQLAlchemy数据库引擎                                                                  |
| `embedder`   | `Optional[Embedder]`             | `None`            | 用于创建嵌入的嵌入器实例（未提供时默认使用OpenAIEmbedder）                            |
| `distance`   | `Distance`                       | `Distance.cosine` | 向量比较的距离度量                                                                    |
| `index`      | `Optional[Union[Ivfflat, HNSW]]` | `HNSW()`          | 向量索引配置                                                                          |
