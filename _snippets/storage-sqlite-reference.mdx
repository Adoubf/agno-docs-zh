### 参数

| 参数         | 类型             | 描述                   | 默认值       |
|--------------|------------------|------------------------|-------------|
| `table_name` | `str`  | SQLite表名称           | 必填        |
| `db_url` | `Optional[str]`  | SQLite连接URL          | `None` |
| `db_file` | `Optional[str]`  | SQLite数据库文件路径   | `None` |
| `db_engine` | `Optional[Engine]`  | 预配置的SQLAlchemy引擎 | `None` |
| `schema_version` | `int` | 模式版本号             | `1` |
| `auto_upgrade_schema` | `bool` | 自动升级模式           | `False` |
