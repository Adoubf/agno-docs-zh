### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于生成嵌入的模型ID | `"voyage-2"` |
| `dimensions` | `int` | 嵌入输出的维度 | `1024` |
| `request_params` | `Optional[Dict[str, Any]]` | 嵌入请求的附加参数 | `None` |
| `api_key` | `Optional[str]` | VoyageAI API密钥 | 环境变量 `VOYAGEAI_API_KEY` |
| `base_url` | `str` | API请求的基础URL | `"https://api.voyageai.com/v1/embeddings"` |
| `max_retries` | `Optional[int]` | 最大重试次数 | `None` |
| `timeout` | `Optional[float]` | 请求超时时间（秒） | `None` |
| `client_params` | `Optional[Dict[str, Any]]` | 客户端初始化的附加参数 | `None` |
| `voyage_client` | `Optional[Client]` | 预配置的VoyageAI客户端 | `None` |
