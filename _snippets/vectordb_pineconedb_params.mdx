| 参数                | 类型                                   | 默认值     | 描述                                                                                   |
| -------------------- | -------------------------------------- | ---------- | -------------------------------------------------------------------------------------- |
| `name`               | `str`                                  | -          | Pinecone索引名称                                                                       |
| `dimension`          | `int`                                  | -          | 嵌入向量的维度                                                                         |
| `spec`               | `Union[Dict, ServerlessSpec, PodSpec]` | -          | 索引规格                                                                               |
| `embedder`           | `Optional[Embedder]`                   | `None`     | 用于创建嵌入的嵌入器实例（未提供时默认使用OpenAIEmbedder）                              |
| `metric`             | `Optional[str]`                        | `"cosine"` | 相似性搜索使用的度量标准                                                               |
| `additional_headers` | `Optional[Dict[str, str]]`             | `None`     | 传递给Pinecone客户端的附加请求头                                                       |
| `pool_threads`       | `Optional[int]`                        | `1`        | Pinecone客户端使用的线程数                                                             |
| `namespace`          | `Optional[str]`                        | `None`     | Pinecone索引的命名空间                                                                 |
| `timeout`            | `Optional[int]`                        | `None`     | Pinecone操作超时时间                                                                   |
| `index_api`          | `Optional[Any]`                        | `None`     | 索引API对象                                                                            |
| `api_key`            | `Optional[str]`                        | `None`     | Pinecone API密钥                                                                       |
| `host`               | `Optional[str]`                        | `None`     | Pinecone主机地址                                                                       |
| `config`             | `Optional[Config]`                     | `None`     | Pinecone配置                                                                           |
| `use_hybrid_search`  | `bool`                                 | `False`    | 是否使用混合搜索                                                                       |
| `hybrid_alpha`       | `float`                                | `0.5`      | 混合搜索的alpha值                                                                      |
