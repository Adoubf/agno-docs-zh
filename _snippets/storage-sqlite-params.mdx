| 参数               | 类型              | 默认值  | 描述                                                     |
| ------------------ | ----------------- | ------ | -------------------------------------------------------- |
| `table_name`      | `str`            | -      | 要使用的表名称。                                         |
| `schema`          | `Optional[str]`  | `"ai"` | 模式名称，默认为"ai"。                                   |
| `db_url`          | `Optional[str]`  | `None` | 数据库URL（如果提供）。                                  |
| `db_engine`       | `Optional[Engine]`  | `None` | 要使用的数据库引擎。                                     |
| `schema_version`   | `int`            | `1`    | 模式版本号，默认为1。                                    |
| `auto_upgrade_schema` | `bool`           | `False` | 若为true，必要时自动升级模式。                           |
