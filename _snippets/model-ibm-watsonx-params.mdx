## 参数

| 参数名 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| id | str | `"ibm/granite-20b-code-instruct"` | 指定使用的模型ID |
| frequency_penalty | float | `None` | 高频token惩罚系数。值越高越抑制重复内容 |
| presence_penalty | float | `None` | 已存在token惩罚系数。值越高越鼓励新话题 |
| max_tokens | int | `None` | 响应中生成的最大token数量 |
| temperature | float | `None` | 控制响应随机性。值越高输出越随机 |
| top_p | float | `None` | 通过核心采样控制响应多样性 |
| logprobs | int | `None` | 返回对数概率的数量 |
| top_logprobs | int | `None` | 返回对数概率的最可能token数量 |
| response_format | Any | `None` | 响应格式规范 |
| api_key | str | `None` | IBM WatsonX API密钥 |
| project_id | str | `None` | IBM WatsonX项目ID |
| url | str | `"https://eu-de.ml.cloud.ibm.com"` | IBM WatsonX API端点URL |
| verify | bool | `True` | 是否验证SSL证书 |
| client_params | Dict[str, Any] | `None` | 传递给客户端的附加参数 |
