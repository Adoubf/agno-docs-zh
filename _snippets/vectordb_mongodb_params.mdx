| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `collection_name` | `str` | MongoDB集合名称 | 必填 |
| `db_url` | `Optional[str]` | MongoDB连接字符串 | `"mongodb://localhost:27017/"` |
| `database` | `str` | 数据库名称 | `"agno"` |
| `embedder` | `Optional[Embedder]` | 用于生成嵌入的嵌入器实例 | `OpenAIEmbedder()` |
| `distance_metric` | `str` | 相似性距离度量标准 | `Distance.cosine` |
| `overwrite` | `bool` | 若为True则覆盖现有集合和索引 | `False` |
| `wait_until_index_ready` | `Optional[float]` | 等待索引就绪的超时时间（秒） | `None` |
| `wait_after_insert` | `Optional[float]` | 插入文档后的等待时间（秒） | `None` |
| `max_pool_size` | `int` | 连接池的最大连接数 | `100` |
| `retry_writes` | `bool` | 是否重试写入操作 | `True` |
| `client` | `Optional[MongoClient]` | 现有MongoClient实例 | `None` |
