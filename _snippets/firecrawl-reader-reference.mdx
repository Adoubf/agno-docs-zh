| 参数 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `api_key` | `Optional[str]` | `None` | 用于身份验证的 Firecrawl API 密钥 |
| `params` | `Optional[Dict]` | `None` | 传递给 Firecrawl API 的附加参数 |
| `mode` | `Literal["scrape", "crawl"]` | `"scrape"` | 操作模式 - "scrape" 表示单页抓取，"crawl" 表示多页抓取 |
| `url` | `str` | 必填 | 要抓取或爬取的网站 URL |
