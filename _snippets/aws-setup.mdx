## AWS 配置

<Steps>
  <Step title="更新凭据">
    在 AWS 上运行需要满足以下任一条件：

    1. 具有 AWS 凭证的 `~/.aws/credentials` 文件
    2. 或 `AWS_ACCESS_KEY_ID` + `AWS_SECRET_ACCESS_KEY` 环境变量

    <Note>

    要创建凭证文件，请安装 [aws cli](https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html) 并运行 `aws configure`

    </Note>

  </Step>
  <Step title="更新区域和子网">
    更新 `aws_region` 并向 `workspace/settings.py` 文件添加 2 个 [子网](https://us-east-1.console.aws.amazon.com/vpc/home?#subnets:)（ECS 服务必需）

    ```python workspace/settings.py
    ws_settings = WorkspaceSettings(
        ...
        # -*- AWS 设置
        # 在此处添加您的子网ID
        subnet_ids=["subnet-xyz", "subnet-xyz"],
        ...
    )
    ```

    <Note>

    请确保子网属于所选的 `aws_region`

    </Note>

  </Step>
</Steps>
