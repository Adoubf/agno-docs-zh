| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `wcd_url` | `Optional[str]` | `None` | Weaviate云端部署的URL。若未提供则回退至`WCD_URL`环境变量。 |
| `wcd_api_key` | `Optional[str]` | `None` | Weaviate云端部署的API密钥。若未提供则回退至`WCD_API_KEY`环境变量。 |
| `client` | `Optional[weaviate.WeaviateClient]` | `None` | 预配置的Weaviate客户端实例。 |
| `local` | `bool` | `False` | 是否使用本地Weaviate实例替代云端服务。 |
| `collection` | `str` | `"default"` | Weaviate中使用的集合名称。 |
| `vector_index` | `VectorIndex` | `VectorIndex.HNSW` | 向量索引类型（可选HNSW、FLAT或DYNAMIC）。 |
| `distance` | `Distance` | `Distance.COSINE` | 向量相似性距离度量标准（如COSINE、DOT等）。 |
| `embedder` | `Optional[Embedder]` | `None` | 用于生成向量嵌入的嵌入器。未提供时默认使用OpenAIEmbedder。 |
| `search_type` | `SearchType` | `SearchType.vector` | 搜索类型（可选vector、keyword或hybrid）。 |
| `reranker` | `Optional[Reranker]` | `None` | 可选的重排序器，用于提升搜索结果质量。 |
| `hybrid_search_alpha` | `float` | `0.5` | 混合搜索中向量与关键词搜索的权重（0.0=仅关键词，1.0=仅向量）。 |
