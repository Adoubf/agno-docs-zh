| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `wcd_url` | `Optional[str]` | Weaviate云服务URL（或使用WCD_URL环境变量） | `None` |
| `wcd_api_key` | `Optional[str]` | Weaviate云服务API密钥（或使用WCD_API_KEY环境变量） | `None` |
| `client` | `Optional[weaviate.WeaviateClient]` | 预配置的Weaviate客户端 | `None` |
| `local` | `bool` | 是否使用本地Weaviate实例 | `False` |
| `collection` | `str` | Weaviate集合名称 | `"default"` |
| `vector_index` | `VectorIndex` | 向量索引类型（HNSW, FLAT, DYNAMIC） | `VectorIndex.HNSW` |
| `distance` | `Distance` | 距离度量方式（COSINE, DOT等） | `Distance.COSINE` |
| `embedder` | `Optional[Embedder]` | 用于生成嵌入的嵌入器 | `OpenAIEmbedder()` |
| `search_type` | `SearchType` | 搜索类型（vector向量, keyword关键词, hybrid混合） | `SearchType.vector` |
| `reranker` | `Optional[Reranker]` | 用于优化搜索结果的重新排序器 | `None` |
| `hybrid_search_alpha` | `float` | 混合搜索的权重因子 | `0.5` |
