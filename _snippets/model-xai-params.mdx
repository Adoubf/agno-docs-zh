| 参数        | 类型            | 默认值                     | 描述                                                                                                               |
| ---------- | --------------- | -------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| `id`       | `str`           | `"grok-beta"`              | 用于生成响应的特定模型ID。                                                                                        |
| `name`     | `str`           | `"xAI"`                    | xAI智能体的名称标识符。                                                                                           |
| `provider` | `str`           | `"xAI"`                    | 模型提供商，由"xAI"与模型ID组合而成。                                                                             |
| `api_key`  | `Optional[str]` | -                          | 用于xAI服务请求认证的API密钥。从环境变量`XAI_API_KEY`中获取。                                                  |
| `base_url` | `str`           | `"https://api.xai.xyz/v1"` | xAI服务的API请求基地址。                                                                                          |
