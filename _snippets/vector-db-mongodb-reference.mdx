| 参数 | 类型 | 默认值 | 描述 |
| --------- | ---- | ------- | ----------- |
| `collection_name` | `str` | 必填 | 用于存储向量和元数据的MongoDB集合名称 |
| `db_url` | `Optional[str]` | `"mongodb://localhost:27017/"` | MongoDB连接字符串 |
| `database` | `str` | `"ai"` | MongoDB数据库名称 |
| `embedder` | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入的嵌入器实例 |
| `distance_metric` | `str` | `Distance.cosine` | 相似性搜索使用的距离度量标准 |
| `overwrite` | `bool` | `False` | 是否覆盖现有集合和索引 |
| `cosmos_compatibility` | `bool` | `False` | 是否启用对Azure Cosmos DB MongoDB vCore的支持 |
| `wait_until_index_ready_in_seconds` | `Optional[float]` | `None` | 等待索引就绪的时间（秒） |
| `wait_after_insert_in_seconds` | `Optional[float]` | `None` | 插入文档后的等待时间（秒） |
