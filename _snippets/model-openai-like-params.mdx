| 参数                | 类型     | 默认值         | 描述                                                |
| ----------------------- | -------- | --------------- | ---------------------------------------------------------- |
| `id`                    | `str`    | `"not-provided"`| 用于生成响应的模型名称。 |
| `name`                  | `str`    | `"OpenAILike"`  | 该聊天模型实例的名称。                      |
| `api_key`               | `str`    | `"not-provided"`| 用于服务请求认证的API密钥。    |
| `override_system_role`  | `bool`   | `False`         | 是否覆盖默认系统角色。               |
| `system_message_role`   | `str`    | `"system"`      | 系统消息所使用的角色。                      |
