| 参数       | 类型                | 默认值                     | 描述                                                                 |

| model      | str                 | "rerank-multilingual-v3.0" | 用于重排序的 Cohere 模型。                                           |

| cohere_client | Optional[CohereClient] | None                  | 可选预配置的 Cohere 客户端实例。                                      |
