| 参数               | 类型       | 默认值           | 描述                                                 |
| ------------------- | ---------- | ---------------- | ---------------------------------------------------- |
| `collection`        | `str`      | -                | 要使用的集合名称。                                   |
| `embedder`          | `Embedder` | OpenAIEmbedder() | 用于嵌入文档内容的嵌入器。                           |
| `distance`          | `Distance` | cosine           | 要使用的距离度量方法。                               |
| `path`              | `str`      | "tmp/chromadb"   | ChromaDB数据的存储路径。                             |
| `persistent_client` | `bool`     | False            | 是否使用持久化的ChromaDB客户端。                     |
