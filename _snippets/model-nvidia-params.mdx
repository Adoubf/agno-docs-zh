| 参数        | 类型            | 默认值                                    | 描述                                                                                                              |
| ---------- | --------------- | ------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------ |
| `id`       | `str`           | `"meta/llama-3.3-70b-instruct"`            | 用于生成响应的特定模型ID。                                                                     |
| `name`     | `str`           | `"Nvidia"`                                 | Nvidia智能体的名称标识符。                                                                                |
| `provider` | `str`           | `"Nvidia" + id`                                           | 模型提供商，由"Nvidia"与模型ID组合而成。                                                         |

| `base_url` | `str`           | `"https://integrate.api.nvidia.com/v1"`    | 向Nvidia服务发起API请求的基础URL。                                                              |
