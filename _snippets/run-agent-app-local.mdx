## 本地运行智能体应用

将`cd`放入`agent-app`文件夹

```bash
cd agent-app
```

使用以下命令启动您的智能体应用：

<CodeGroup>

```bash 终端
ag ws up
```

```bash 速记
ag ws up dev:docker
```

```bash 完整选项
ag ws up --env dev --infra docker
```

</CodeGroup>

**按Enter键**确认并等待几秒下载镜像（仅首次运行）。通过docker仪表盘查看容器状态和日志。

- 访问[localhost:8501](http://localhost:8501)查看streamlit用户界面
- 访问[localhost:8000/docs](http://localhost:8000/docs)查看FastAPI路由

![智能体应用界面](/images/agent-app-ui.png)

注意事项：

- `Agents`定义在`agents`文件夹中
- streamlit应用定义在`ui`文件夹
- API路由定义在`api`文件夹
