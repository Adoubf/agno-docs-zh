| 参数         | 类型              | 默认值                     | 描述                                                                                                             |
| ------------ | ----------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------- |
| `id`       | `str`           | `"v0-1.0-md"`              | 用于生成响应的特定模型ID。                                                                                      |
| `name`     | `str`           | `"v0"`                    | v0智能体的名称标识符。                                                                                          |
| `provider` | `str`           | `"Vercel"`                 | 模型供应商名称，由"v0"与模型ID组合而成。                                                                        |
| `api_key`  | `Optional[str]` | -                          | 用于验证v0服务请求的API密钥。从环境变量`V0_API_KEY`中获取。                                                 |
| `base_url` | `str`           | `"https://api.v0.dev/v1/"` | v0服务的API请求基础URL。                                                                                        |
