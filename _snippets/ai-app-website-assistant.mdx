### Website Assistant

- Click on **Website Assistant** in the sidebar and add a domain you'd like to add to the knowledge base. For example: `https://docs.phidata.com/introduction`

- Ask questions like `what is phidata?`

<Tip>
  If you run into issues, check the docker dashboard for logs or message us
</Tip>
