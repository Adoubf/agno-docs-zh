## 配置

<Steps>
  <Step title="创建并激活虚拟环境">

    <CodeGroup>
    ```bash Mac
    python3 -m venv .venv
    source .venv/bin/activate
    ```

    ```bash Windows
    python3 -m venv aienv
    aienv/scripts/activate
    ```
    </CodeGroup>

  </Step>
  <Step title="安装 Agno">

    <CodeGroup>
    ```bash Mac
    pip install -U "agno[aws]"
    ```

    ```bash Windows
    pip install -U "agno[aws]"
    ```
    </CodeGroup>

  </Step>
  <Step title="安装 uv 和 docker">
    - 安装 [uv](https://docs.astral.sh/uv/#getting-started) 用于管理 Python 环境

    ```bash
    curl -LsSf https://astral.sh/uv/install.sh | sh
    ```

    - 安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/) 以便本地运行应用
  </Step>
  <Step title="导出您的 OpenAI 密钥">

    <CodeGroup>
    ```bash Mac（苹果电脑）
    export OPENAI_API_KEY=sk-***
    ```

    ```bash Windows
    setx OPENAI_API_KEY sk-***
    ```
    </CodeGroup>

    <Tip>
    Agno 兼容任何模型提供商，只需更新 workspace 中的智能体即可。
    </Tip>
  </Step>
</Steps>
