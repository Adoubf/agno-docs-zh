## 更新密钥

<Steps>
  <Step title="RDS数据库密码">
    在 `workspace/secrets/prd_db_secrets.yml` 中更新 RDS 数据库密码

    ```python workspace/secrets/prd_db_secrets.yml
    # RDS 数据库使用的密钥
    MASTER_USERNAME: app
    MASTER_USER_PASSWORD: "app9999!!"
    ```

  </Step>
  <Step title="应用密钥">
    将应用使用的其他密钥添加到 `workspace/secrets/prd_app_secrets.yml`

    ```python workspace/secrets/prd_app_secrets.yml
    SECRET_KEY: "django-insecure-...."
    # OPENAI_API_KEY: "sk-***"
    ```

  </Step>
</Steps>
