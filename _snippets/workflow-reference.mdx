| 参数          | 类型                                  | 默认值                                | 描述                                                                                                                              |
|---------------|---------------------------------------|----------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| `name`        | `Optional[str]`                       | `None`                                 | **工作流名称**                                                                                                                        |
| `workflow_id` | `Optional[str]`                       | `None`                                 | **工作流UUID**（若未设置则自动生成）                                                                                             |
| `description` | `Optional[str]`                       | `None`                                 | **工作流描述**（仅在UI中显示）                                                                                          |
| `user_id`     | `Optional[str]`                       | `None`                                 | **用户ID**，与当前工作流交互的用户                                                                                        |
| `session_id`  | `Optional[str]`                       | `None`                                 | **会话UUID**（若未设置则自动生成）                                                                                              |
| `session_name`| `Optional[str]`                       | `None`                                 | **会话名称**                                                                                                                         |
| `session_state`| `Dict[str, Any]`                     | `{}` (空字典)                      | **会话状态**，存储在数据库中                                                                                                 |
| `memory`      | `Optional[WorkflowMemory]`            | `None`                                 | **工作流记忆**                                                                                                                      |
| `storage`     | `Optional[WorkflowStorage]`           | `None`                                 | **工作流存储**                                                                                                                     |
| `extra_data`  | `Optional[Dict[str, Any]]`            | `None`                                 | **额外数据**，与工作流关联存储                                                                                                 |
| `debug_mode`  | `bool`                                | `False`                                | 启用调试日志                                                                                                                        |
| `monitoring`  | `bool`                                | `False` (环境变量: `AGNO_MONITOR`)          | 若为`True`，则将工作流信息日志发送至agno.com进行监控。默认值为`True`，若环境变量中存在`AGNO_MONITOR="true"`。         |
| `telemetry`   | `bool`                                | `True` (环境变量: `AGNO_TELEMETRY`)         | 若为`True`，则记录最小化遥测数据用于分析。默认值为`True`，若环境变量中存在`AGNO_TELEMETRY="true"`。                       |
| `run_id`      | `Optional[str]`                       | `None`                                 | **（请勿手动设置）** 每次工作流运行的唯一ID                                                                                |
| `run_input`   | `Optional[Dict[str, Any]]`            | `None`                                 | **（请勿手动设置）** 传递给当前运行的输入                                                                                |
| `run_response`| `Optional[RunResponse]`               | `None`                                 | **（请勿手动设置）** 当前运行生成的响应                                                                          |
| `images`      | `Optional[List[ImageArtifact]]`       | `None`                                 | **会话期间生成的图像**                                                                                                 |
| `videos`      | `Optional[List[VideoArtifact]]`       | `None`                                 | **会话期间生成的视频**                                                                                                 |
| `audio`       | `Optional[List[AudioArtifact]]`       | `None`                                 | **会话期间生成的音频**                                                                                                  |
