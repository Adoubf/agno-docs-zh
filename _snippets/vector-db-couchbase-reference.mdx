| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `bucket_name` | `str` | Couchbase 桶名称 | 必填 |
| `scope_name` | `str` | 桶内作用域名称 | 必填 |
| `collection_name` | `str` | 作用域内集合名称 | 必填 |
| `couchbase_connection_string` | `str` | Couchbase 集群连接字符串 | 必填 |
| `cluster_options` | `ClusterOptions` | 配置 Couchbase 集群连接的选项 | 必填 |
| `search_index` | `Union[str, SearchIndex]` | 搜索索引配置（可接受索引名称或 SearchIndex 定义） | 必填 |
| `embedder` | `Embedder` | 用于生成嵌入的嵌入器实例 | `OpenAIEmbedder()` |
| `overwrite` | `bool` | 是否覆盖现有集合 | `False` |
| `is_global_level_index` | `bool` | 搜索索引是否为全局级别 | `False` |
| `wait_until_index_ready` | `Optional[float]` | 等待索引准备就绪的超时时间（秒） | `None` |
| `batch_limit` | `int` | 单批次处理的最大文档数（适用于同步和异步操作） | `500` |
