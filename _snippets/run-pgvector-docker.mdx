## 在 Docker 上运行 PgVector

创建一个名为 `resources.py` 的文件，内容如下：

```python resources.py
from agno.docker.app.postgres import PgVectorDb
from agno.docker.resources import DockerResources

# -*- PgVector 运行在端口 5432:5432
vector_db = PgVectorDb(
    pg_user="ai",
    pg_password="ai",
    pg_database="ai",
    debug_mode=True,
)

# -*- Docker资源
dev_docker_resources = DockerResources(apps=[vector_db])
```

使用以下命令启动资源：

<CodeGroup>

```bash 苹果
ag start resources.py
```

```bash Windows
ag start resources.py
```

</CodeGroup>

**按 Enter 键**确认并在 Docker 仪表盘中验证容器状态。
