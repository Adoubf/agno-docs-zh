| 参数        | 类型     | 默认值   | 描述                                                                 |
|-------------|----------|----------|---------------------------------------------------------------------|
| `model` | `str` | -        | 指定使用的模型ID。                                                   |
| `name` | `str` | -        | 该大语言模型的名称。注意：此名称不会发送至LLM API。                  |
| `metrics` | `Dict[str, Any]` | -        | 为该大语言模型收集的指标。注意：此数据不会发送至LLM API。             |
| `response_format` | `Any` | -        | 响应格式。                                                           |
| `tools` | `List[Union[Tool, Dict]]` | -        | 提供给大语言模型的工具列表。工具是模型可能生成JSON输入的函数。若提供字典形式，则不会被模型调用。务必使用`add_tool()`方法添加工具。 |
| `tool_choice` | `Union[str, Dict[str, Any]]` | -        | 控制模型调用哪些（如有）函数。`"none"`表示模型仅生成消息而不调用函数。`"auto"`表示模型可选择生成消息或调用函数。通过`{"type": "function", "function": {"name": "my_function"}}`指定特定函数将强制模型调用该函数。未配置工具时默认值为`"none"`，配置工具时默认值为`"auto"`。 |
| `run_tools` | `bool` | `True` | 若为True，则执行工具。                                               |
| `show_tool_calls` | `bool` | -        | 若为True，响应中将显示工具调用。                                     |
| `functions` | `Dict[str, Function]` | -        | 从工具中提取的函数。注意：这些不会发送至LLM API，仅用于执行。        |
| `function_call_limit` | `int` | `20` | 允许的最大函数调用次数。                                             |
| `function_call_stack` | `List[FunctionCall]` | -        | 函数调用堆栈。                                                      |
| `system_prompt` | `str` | -        | 提供给大语言模型的系统提示。                                        |
| `instructions` | `str` | -        | 提供给大语言模型的指令。                                            |
