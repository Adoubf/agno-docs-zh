## 使用FastAPI部署智能体

[FastAPI](https://fastapi.tiangolo.com/) 是构建REST API的卓越框架。它快速、设计精良且深受用户喜爱。大多数生产级应用都采用像[next.js](https://nextjs.org)这样的前端框架配合REST API构建，这正是FastAPI的优势所在。

您的代码库已预配置好[FastAPI](https://fastapi.tiangolo.com/)和[PostgreSQL](https://www.postgresql.org/)，并包含示例路由。使用以下命令启动工作区：

<CodeGroup>

```bash 终端
ag ws up
```

```bash 速记
ag ws up dev:docker
```

</CodeGroup>

**按Enter键**确认启动，首次运行时需等待镜像下载（仅首次需要）。通过docker仪表盘查看容器状态和日志。

- 访问[localhost:8000/docs](http://localhost:8000/docs)查看API端点文档
- 使用以下命令测试`/v1/agents/{agent_id}/run`端点：

```json
{
  "message": "howdy",
  "agent_id": "sage",
  "stream": true
}
```
