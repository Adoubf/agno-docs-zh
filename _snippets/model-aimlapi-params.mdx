| 参数          | 类型             | 默认值                          | 描述                                                                                                                           |
| ------------ | --------------- | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `id`         | `str`           | `"gpt-4o-mini"`                  | 用于生成响应的特定模型ID。                                                                                                     |
| `name`       | `str`           | `"AIMLApi"`                      | AI/ML API智能体的名称标识符。                                                                                                  |
| `provider`   | `str`           | `"AIMLApi:" + id`                | 模型提供商，由`"AIMLApi"`与模型ID组合而成。                                                                                 |
| `api_key`    | `Optional[str]` | –                                | 用于验证AI/ML API服务请求的API密钥。从环境变量`AIMLAPI_API_KEY`中获取。                                                        |
| `base_url`   | `str`           | `"https://api.aimlapi.com/v1"`   | 向AI/ML API服务发起请求的基地址URL。                                                                                          |
| `max_tokens` | `int`           | `4096`                           | 响应中生成的最大令牌数。                                                                                                       |
