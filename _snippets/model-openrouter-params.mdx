| 参数         | 类型            | 默认值                          | 描述                                                                                                                      |
| ------------ | --------------- | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `id`         | `str`           | `"gpt-4o"`                       | 用于生成响应的特定模型ID。                                                                             |
| `name`       | `str`           | `"OpenRouter"`                   | OpenRouter智能体的名称标识符。                                                                                    |
| `provider`   | `str`           | `"OpenRouter:"+id`               | 模型提供商，由"OpenRouter"与模型ID组合而成。                                                             |
| `api_key`    | `Optional[str]` | -                                | 用于向OpenRouter服务认证请求的API密钥。从环境变量`OPENROUTER_API_KEY`获取。 |
| `base_url`   | `str`           | `"https://openrouter.ai/api/v1"` | 向OpenRouter服务发起API请求的基础URL。                                                                  |
| `max_tokens` | `int`           | `1024`                           | 响应中生成的最大token数量。                                                                        |
