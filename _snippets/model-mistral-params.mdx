| 参数               | 类型                                                      | 默认值                 | 描述                                                                                                                                                                                         |
| ------------------- | --------------------------------------------------------- | ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `id`                | `str`                                                     | `"mistral-large-latest"` | 模型ID。                                                                                                                                                                                    |
| `name`              | `str`                                                     | `"MistralChat"`          | 模型名称。                                                                                                                                                                                 |
| `provider`          | `str`                                                     | `"Mistral"`              | 模型提供商。                                                                                                                                                                               |
| `temperature`       | `Optional[float]`                                         | `None`                   | 控制输出生成的随机性。                                                                                                                                                                     |
| `max_tokens`        | `Optional[int]`                                           | `None`                   | 生成的最大token数量。                                                                                                                                                                      |
| `top_p`             | `Optional[float]`                                         | `None`                   | 控制输出生成的多样性。                                                                                                                                                                     |
| `random_seed`       | `Optional[int]`                                           | `None`                   | 随机数生成种子。                                                                                                                                                                           |
| `safe_mode`         | `bool`                                                    | `False`                  | 启用内容过滤功能。                                                                                                                                                                         |
| `safe_prompt`       | `bool`                                                    | `False`                  | 对提示内容应用内容过滤。                                                                                                                                                                   |
| `response_format`   | `Optional[Union[Dict[str, Any], ChatCompletionResponse]]` | `None`                   | 指定期望的响应格式。                                                                                                                                                                       |
| `request_params`    | `Optional[Dict[str, Any]]`                                | `None`                   | 额外的请求参数。                                                                                                                                                                           |
| `api_key`           | `Optional[str]`                                           | `None`                   | 您的Mistral API密钥。                                                                                                                                                                      |
| `endpoint`          | `Optional[str]`                                           | `None`                   | 自定义API端点URL。                                                                                                                                                                         |
| `max_retries`       | `Optional[int]`                                           | `None`                   | API调用最大重试次数。                                                                                                                                                                      |
| `timeout`           | `Optional[int]`                                           | `None`                   | API调用超时时间（秒）。                                                                                                                                                                    |
| `client_params`     | `Optional[Dict[str, Any]]`                                | `None`                   | 额外的客户端参数。                                                                                                                                                                         |
| `mistral_client`    | `Optional[MistralClient]`                                 | `None`                   | 自定义Mistral客户端实例。                                                                                                                                                                  |
| `store`             | `Optional[bool]`                                          | `None`                   | 是否存储本次聊天补全请求的输出，用于模型蒸馏或评估产品。                                                                                                                                   |
| `frequency_penalty` | `Optional[float]`                                         | `None`                   | 取值范围-2.0到2.0。正值会根据token在现有文本中的出现频率进行惩罚，降低模型逐字重复同一内容的概率。                                                                                         |
| `logit_bias`        | `Optional[Any]`                                           | `None`                   | 通过将token ID映射到-100至100之间的偏置值，修改特定token在补全结果中出现概率的JSON对象。                                                                                                   |
| `logprobs`          | `Optional[bool]`                                          | `None`                   | 是否返回输出token的对数概率。                                                                                                                                                              |
| `presence_penalty`  | `Optional[float]`                                         | `None`                   | 取值范围-2.0到2.0。正值会根据token是否在现有文本中出现过进行惩罚，增加模型讨论新话题的概率。                                                                                               |
| `stop`              | `Optional[Union[str, List[str]]]`                         | `None`                   | 最多4个序列，当API生成到这些序列时将停止生成后续token。                                                                                                                                     |
| `top_logprobs`      | `Optional[int]`                                           | `None`                   | 为每个生成token返回的最高对数概率数量。                                                                                                                                                     |
| `user`              | `Optional[str]`                                           | `None`                   | 代表终端用户的唯一标识符，有助于监测和检测滥用行为。                                                                                                                                       |
