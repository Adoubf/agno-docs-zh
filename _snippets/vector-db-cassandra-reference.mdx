| 参数        | 类型                | 默认值           | 描述                                                           |
| ----------- | ------------------- | ---------------- | -------------------------------------------------------------- |
| `table_name` | `str`               | `None`                | 用于存储向量和元数据的 Cassandra 表名称                        |
| `keyspace`   | `str`               | `None`                | 将在其中创建表的 Cassandra 键空间名称                          |
| `embedder`   | `Optional[Embedder]` | `OpenAIEmbedder()` | 用于生成嵌入向量的嵌入器实例                                   |
| `session`    | `CassandraSession`  | `None`                | 用于数据库操作的活跃 Cassandra 会话对象                        |
