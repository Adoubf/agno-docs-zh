| 参数          | 类型           | 默认值  | 描述                                                                                                                   |
| ------------- | -------------- | ------- | ---------------------------------------------------------------------------------------------------------------------- |
| `uri`         | `str`          | -       | 连接目标URI地址                                                                                                        |
| `table`       | `LanceTable`   | -       | 使用的Lance数据表                                                                                                      |
| `table_name`  | `str`          | -       | 使用的数据表名称                                                                                                       |
| `connection`  | `DBConnection` | -       | 使用的数据库连接                                                                                                       |
| `api_key`     | `str`          | -       | 使用的API密钥                                                                                                          |
| `embedder`    | `Embedder`     | -       | 使用的嵌入模型                                                                                                         |
| `search_type` | `SearchType`   | vector  | 使用的搜索类型                                                                                                         |
| `distance`    | `Distance`     | cosine  | 使用的距离计算方法                                                                                                     |
| `nprobes`     | `int`          | -       | 使用的探测点数。[更多信息](https://lancedb.github.io/lancedb/ann_indexes/#use-gpu-to-build-vector-index)               |
| `reranker`    | `Reranker`     | -       | 使用的重排序器。[更多信息](https://lancedb.github.io/lancedb/hybrid_search/eval/)                                      |
| `use_tantivy` | `bool`         | -       | 是否启用tantivy                                                                                                        |
