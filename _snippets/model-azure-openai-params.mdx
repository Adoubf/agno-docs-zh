| 参数                 | 类型                          | 默认值             | 说明                                                                  |
| ------------------------- | ----------------------------- | ------------------- | ---------------------------------------------------------------------------- |
| `id`                      | `str`                         | -                   | 用于生成响应的特定模型ID。该字段为必填项。 |
| `name`                    | `str`                         | `"AzureOpenAI"`     | 智能体的名称标识符。                                           |
| `provider`                | `str`                         | `"Azure"`           | 模型的提供商。                                                   |
| `api_key`                 | `Optional[str]`               | `"None"`            | 用于认证Azure OpenAI服务请求的API密钥。         |
| `api_version`             | `str`                         | `"2024-10-21"`      | 所使用的Azure OpenAI API版本。                                  |
| `azure_endpoint`          | `Optional[str]`               | `"None"`            | Azure OpenAI服务的端点URL。                               |
| `azure_deployment`        | `Optional[str]`               | `"None"`            | Azure中的部署名称或ID。                                          |
| `azure_ad_token`          | `Optional[str]`               | `"None"`            | 用于认证请求的Azure Active Directory令牌。                |
| `azure_ad_token_provider` | `Optional[Any]`               | `"None"`            | 获取Azure Active Directory令牌的提供商。                    |
| `openai_client`           | `Optional[AzureOpenAIClient]` | `"None"`            | 用于发起API请求的AzureOpenAIClient实例。           |
