## 本地运行你的 Agent Api

`cd` 进入 `agent-api` 文件夹

```bash
cd agent-api
```

使用以下命令启动你的 Agent Api：

<CodeGroup>

```bash 终端
ag ws up
```

```bash 速记
ag ws up dev:docker
```

```bash 完整选项
ag ws up --env dev --infra docker
```

</CodeGroup>

**按回车键**确认并等待几秒下载镜像（仅首次运行）。在 Docker 仪表盘中检查容器状态并查看日志。

- 访问 [localhost:8000/docs](http://localhost:8000/docs) 查看 FastAPI 路由。

注意事项：

- `Agents` 定义在 `agents` 文件夹中。
- API 路由定义在 `api` 文件夹中。
