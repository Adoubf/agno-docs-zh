## 可选：运行 Jupyterlab

Jupyter notebook 是 AI 开发必备工具，您的 `ai-app` 已预装了所需依赖的笔记本环境。启动方式如下：

<Steps>
  <Step title="启用 Jupyter">
    更新 `workspace/settings.py` 文件并设置 `dev_jupyter_enabled=True`

    ```python workspace/settings.py
    ...
    ws_settings = WorkspaceSettings(
        ...
        # 取消以下行的注释
        dev_jupyter_enabled=True,
    ...
    ```

  </Step>
  <Step title="启动 Jupyter">

    <CodeGroup>

    ```bash 终端
    ag ws up --group jupyter
    ```

    ```bash 速记
    ag ws up dev:docker:jupyter
    ```

    </CodeGroup>

    **按回车键**确认，首次运行需等待镜像下载（仅首次）。可通过 Docker 仪表盘查看容器状态和日志。

  </Step>
  <Step title="查看 JupyterLab 界面">
    - 访问 [localhost:8888](http://localhost:8888) 进入 Jupyterlab 界面。密码：**admin**
    - 可在 `notebooks` 文件夹中体验示例手册。

    ![Jupyter Notebook](/images/ai-app-jupyter-local.png)

  </Step>
</Steps>
