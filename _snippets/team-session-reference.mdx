| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `session_id` | `str` | 必填 | 会话 UUID |
| `team_id` | `Optional[str]` | `None` | 该会话关联的团队 ID |
| `user_id` | `Optional[str]` | `None` | 与该团队交互的用户 ID |
| `memory` | `Optional[Dict[str, Any]]` | `None` | 团队记忆 |
| `team_data` | `Optional[Dict[str, Any]]` | `None` | 团队数据：team_id、名称和模型 |
| `session_data` | `Optional[Dict[str, Any]]` | `None` | 会话数据：会话名称、会话状态、图片、视频、音频 |
| `extra_data` | `Optional[Dict[str, Any]]` | `None` | 与该团队关联的额外存储数据 |
| `created_at` | `Optional[int]` | `None` | 该会话创建的 Unix 时间戳 |
| `updated_at` | `Optional[int]` | `None` | 该会话最后更新的 Unix 时间戳 |
