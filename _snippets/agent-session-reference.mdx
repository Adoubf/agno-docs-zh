| 参数 | 类型 | 默认值 | 描述 |
|-----------|------|---------|-------------|
| `session_id` | `str` | 必填 | 会话UUID |
| `agent_id` | `Optional[str]` | `None` | 与该会话关联的智能体ID |
| `user_id` | `Optional[str]` | `None` | 与该智能体交互的用户ID |
| `team_session_id` | `Optional[str]` | `None` | 该会话可能关联的团队会话ID |
| `memory` | `Optional[Dict[str, Any]]` | `None` | 智能体记忆 |
| `agent_data` | `Optional[Dict[str, Any]]` | `None` | 智能体数据：agent_id、名称和模型 |
| `session_data` | `Optional[Dict[str, Any]]` | `None` | 会话数据：会话名称、会话状态、图像、视频、音频 |
| `extra_data` | `Optional[Dict[str, Any]]` | `None` | 与该智能体一起存储的额外数据 |
| `created_at` | `Optional[int]` | `None` | 该会话创建的Unix时间戳 |
| `updated_at` | `Optional[int]` | `None` | 该会话最后更新的Unix时间戳 |
