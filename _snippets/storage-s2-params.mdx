| 参数                  | 类型               | 默认值  | 描述                                                         |
| --------------------- | ------------------ | ------- | ------------------------------------------------------------ |
| `table_name`          | `str`              | -       | 要使用的表名。                                               |
| `schema`              | `Optional[str]`    | `"ai"`  | 模式名称。                                                   |
| `db_url`              | `Optional[str]`    | `None`  | 数据库URL（若提供）。                                        |
| `db_engine`           | `Optional[Engine]` | `None`  | 要使用的数据库引擎。                                         |
| `schema_version`      | `int`              | `1`     | 模式版本。                                                   |
| `auto_upgrade_schema` | `bool`             | `False` | 若为`true`，则在必要时自动升级模式。               |
