| 字段 | 描述 |
|-------------------------|-----------------------------------------------------------------------------|
| input_tokens | 输入给模型的提示/输入中的token数量。 |
| output_tokens | 模型生成输出中的token数量。 |
| total_tokens | 使用的总token数（输入+输出）。 |
| prompt_tokens | 提示中的token数（对于OpenAI等同于input_tokens）。 |
| completion_tokens | 完成文本中的token数（对于OpenAI等同于output_tokens）。 |
| audio_tokens | 音频总token数（若使用音频输入/输出）。 |
| input_audio_tokens | 输入音频中的token数。 |
| output_audio_tokens | 输出音频中的token数。 |
| cached_tokens | 从缓存提供的token数（若启用缓存）。 |
| cache_write_tokens | 写入缓存的token数。 |
| reasoning_tokens | 用于推理步骤的token数（若启用）。 |
| prompt_tokens_details | 包含提示token详细分类的字典（OpenAI专用）。 |
| completion_tokens_details | 包含完成token详细分类的字典（OpenAI专用）。 |
| additional_metrics | 模型/工具提供的额外指标（如延迟、成本等）。 |
| time | 生成消息耗时（秒）。 |
| time_to_first_token | 生成第一个token的耗时（秒）。 |
> 注意：并非所有字段始终存在，具体取决于模型/工具和运行情况。
