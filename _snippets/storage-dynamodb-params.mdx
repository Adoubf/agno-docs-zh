| 参数                       | 类型            | 默认值  | 描述                                              |
| ---------------------------- | --------------- | ------- | ------------------------------------------------ |
| `table_name`                 | `str`           | -       | 要使用的表名称。                                  |
| `region_name`                | `Optional[str]` | `None`  | DynamoDB 表的区域名称。                           |
| `aws_access_key_id`          | `Optional[str]` | `None`  | 可选的 AWS 访问密钥 ID。                          |
| `aws_secret_access_key`      | `Optional[str]` | `None`  | 可选的 AWS 秘密访问密钥。                         |
| `endpoint_url`               | `Optional[str]` | `None`  | 可选的端点 URL。                                  |
| `create_table_if_not_exists` | `bool`          | `True`  | 若为 true，当表不存在时自动创建。                 |
