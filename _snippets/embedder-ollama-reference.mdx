### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `id` | `str` | 用于生成嵌入的模型ID | `"openhermes"` |
| `dimensions` | `int` | 嵌入输出的维度 | `4096` |
| `host` | `Optional[str]` | Ollama服务器的主机URL | `None` |
| `timeout` | `Optional[Any]` | 请求超时时间 | `None` |
| `options` | `Optional[Any]` | 嵌入生成的附加选项 | `None` |
| `client_kwargs` | `Optional[Dict[str, Any]]` | 客户端初始化的附加参数 | `None` |
| `ollama_client` | `Optional[OllamaClient]` | 预配置的Ollama客户端 | `None` |
