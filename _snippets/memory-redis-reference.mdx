### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `prefix` | `str` | Redis键前缀，用于命名空间化记忆存储 | `"agno_memory"` |
| `host` | `str` | Redis主机地址 | `"localhost"` |
| `port` | `int` | Redis端口号 | `6379` |
| `db` | `int` | Redis数据库编号 | `0` |
| `password` | `Optional[str]` | Redis密码（如需认证） | `None` |
