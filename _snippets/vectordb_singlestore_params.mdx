| 参数         | 类型               | 默认值            | 描述                                               |
| ------------ | ------------------ | ------------------ | --------------------------------------------------- |
| `collection` | `str`              | -                  | 要使用的集合名称。                                  |
| `schema`     | `Optional[str]`    | `"ai"`             | 要使用的数据库模式。                                |
| `db_url`     | `Optional[str]`    | `None`             | 数据库连接 URL。                                    |
| `db_engine`  | `Optional[Engine]` | `None`             | SQLAlchemy 引擎实例。                               |
| `embedder`   | `Embedder`         | `OpenAIEmbedder()` | 用于创建向量嵌入的嵌入器。                          |
| `distance`   | `Distance`         | `Distance.cosine`  | 相似性搜索使用的距离度量标准。                      |
