| 参数                | 类型               | 默认值  | 描述                                                     |
| ------------------- | ------------------ | ------- | -------------------------------------------------------- |
| `table_name`        | `str`              | -       | 存储工作流会话的表名称。                                 |
| `schema`            | `Optional[str]`    | `"ai"`  | 表使用的模式。                                           |
| `db_url`            | `Optional[str]`    | `None`  | 连接的数据库URL。                                        |
| `db_engine`         | `Optional[Engine]` | `None`  | 使用的SQLAlchemy数据库引擎。                             |
| `schema_version`    | `int`              | `1`     | 模式版本。                                               |
| `auto_upgrade_schema` | `bool`           | `False` | 是否自动升级模式。                                       |
