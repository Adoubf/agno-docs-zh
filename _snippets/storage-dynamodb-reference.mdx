### 参数

| 参数         | 类型             | 描述                   | 默认值       |
|--------------|------------------|------------------------|-------------|
| `table_name` | `str` | DynamoDB表名称          | 必填        |
| `region_name` | `Optional[str]` | AWS区域名称             | `None` |
| `aws_access_key_id` | `Optional[str]` | AWS访问密钥ID           | `None` |
| `aws_secret_access_key` | `Optional[str]` | AWS秘密访问密钥         | `None` |
| `endpoint_url` | `Optional[str]` | 自定义端点URL           | `None` |
| `create_table_if_not_exists` | `bool` | 缺失时自动创建表        | `True` |
