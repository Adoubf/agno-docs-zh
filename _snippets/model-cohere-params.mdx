| 参数                           | 类型                       | 默认值               | 描述                                                                                                                                                                                       |
| ------------------------------ | -------------------------- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `id`                | `str`            | `"command-r-plus"`       | 用于生成响应的具体模型ID。                                                                                                                                                                 |
| `name`                | `str`            | `"cohere"`       | 智能体的名称标识符。                                                                                                                                                                       |
| `provider`                | `str`            | `"Cohere"`       | 模型供应商名称。                                                                                                                                                                           |
| `temperature`                | `Optional[float]`           | `None`      | 采样温度值，范围在0至2之间。较高值（如0.8）会使输出更随机，较低值（如0.2）会使输出更集中且确定性更强。                                                                                     |
| `max_tokens`               | `Optional[int]`           | `None`      | 响应中生成的最大token数量。                                                                                                                                                                |
| `top_k`               | `Optional[int]`           | `None`      | 保留最高概率词汇token的数量，用于top-k筛选。                                                                                                                                               |
| `top_p`               | `Optional[float]`           | `None`      | 核采样参数。模型会考虑具有top_p概率质量的token结果。                                                                                                                                       |
| `frequency_penalty`               | `Optional[float]`           | `None`      | -2.0至2.0间的数值。正值会根据token当前在文本中的出现频率进行惩罚，从而降低模型逐字重复相同内容的概率。                                                                                     |
| `presence_penalty`               | `Optional[float]`           | `None`      | -2.0至2.0间的数值。正值会根据token是否已在文本中出现过进行惩罚，从而增加模型讨论新话题的概率。                                                                                             |
| `request_params`               | `Optional[Dict[str, Any]]`           | `None`      | 请求中包含的附加参数。                                                                                                                                                                     |
| `add_chat_history`               | `bool`           | `False`      | 是否将聊天历史添加到Cohere消息中（而非使用conversation_id）。                                                                                                                              |
| `api_key`               | `Optional[str]`           | `None`      | 用于Cohere服务请求认证的API密钥。                                                                                                                                                          |
| `client_params`               | `Optional[Dict[str, Any]]`           | `None`      | 客户端配置的附加参数。                                                                                                                                                                     |
| `cohere_client`               | `Optional[CohereClient]`           | `None`      | 预配置的Cohere客户端实例。                                                                                                                                                                 |
| `structured_outputs`               | `bool`           | `False`      | 是否对该模型使用结构化输出。                                                                                                                                                               |
| `supports_structured_outputs`               | `bool`           | `True`      | 该模型是否支持结构化输出。                                                                                                                                                                 |
| `add_images_to_message_content`               | `bool`           | `True`      | 是否在消息内容中添加图像。                                                                                                                                                                 |
| `override_system_role`               | `bool`           | `True`      | 是否覆盖系统角色设置。                                                                                                                                                                     |
| `system_message_role`               | `str`           | `"system"`      | 系统消息映射的目标角色。                                                                                                                                                                   |
