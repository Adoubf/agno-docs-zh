### 参数

| 参数 | 类型 | 描述 | 默认值 |
|-----------|------|-------------|---------|
| `table_name` | `str` | SQLite表名称 | `"memory"` |
| `db_url` | `Optional[str]` | SQLite数据库URL | `None` |
| `db_file` | `Optional[str]` | SQLite数据库文件路径 | `None` |
| `db_engine` | `Optional[Engine]` | 预配置的SQLAlchemy引擎 | `None` |
