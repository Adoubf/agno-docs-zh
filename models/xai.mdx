xAI是一个为大型语言模型提供终端接入的平台。
查看其模型列表[请点击这里](https://docs.x.ai/docs/models)。

我们建议通过实验来寻找最适合您应用场景的模型。`grok-beta`模型适用于大多数使用场景。

xAI的API存在速率限制。更多信息请参阅[文档](https://docs.x.ai/docs/usage-tiers-and-rate-limits)。

## 认证

设置您的`XAI_API_KEY`环境变量。您可以在[xAI控制台](https://console.x.ai/)获取该变量。

<CodeGroup>

```bash Mac
export XAI_API_KEY=sk-***
```

```bash Windows
setx XAI_API_KEY sk-***
```

</CodeGroup>

## 示例

使用`xAI`配合您的`Agent`：

<CodeGroup>

```python agent.py

from agno.agent import Agent, RunResponse
from agno.models.xai import xAI

agent = Agent(
    model=xAI(id="grok-beta"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>
