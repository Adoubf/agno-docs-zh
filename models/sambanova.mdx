Sambanova 是一个为大型语言模型提供端点服务的平台。请注意，目前 Sambanova 不支持函数调用功能。

## 认证

设置您的 `SAMBANOVA_API_KEY` 环境变量。密钥获取地址：[点击此处](https://cloud.sambanova.ai/apis)。

<CodeGroup>

```bash 苹果
export SAMBANOVA_API_KEY=***
```

```bash Windows
setx SAMBANOVA_API_KEY ***
```

</CodeGroup>

## 示例

使用 `Sambanova` 配合您的 `Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.sambanova import Sambanova

agent = Agent(model=Sambanova(), markdown=True)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>
