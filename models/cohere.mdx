利用Cohere强大的指令模型及其他功能。

[Cohere](https://cohere.com)提供广泛的模型系列，特别适合微调应用。查看其模型库请访问[这里](https://docs.cohere.com/v2/docs/models)。

我们建议通过实验找到最适合您用例的模型。以下是通用建议：

- `command` 模型适用于大多数基础用例
- `command-light` 模型适合小型任务和快速推理
- `command-r7b-12-2024` 模型擅长RAG任务、复杂推理和多步骤任务

Cohere还支持模型微调。具体操作指南请参阅[这里](https://docs.cohere.com/v2/docs/fine-tuning)。

Cohere采用分级速率限制机制。详见[文档](https://docs.cohere.com/v2/docs/rate-limits)。

## 认证

设置您的 `CO_API_KEY` 环境变量。API密钥获取地址[在此](https://dashboard.cohere.com/api-keys)。

<CodeGroup>

```bash Mac
export CO_API_KEY=***
```

```bash Windows
setx CO_API_KEY ***
```

</CodeGroup>

## 示例

将 `Cohere` 与您的 `Agent` 配合使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.cohere import Cohere

agent = Agent(
    model=Cohere(id="command-r-08-2024"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 查看更多示例请访问[这里](../examples/models/cohere)。 </Note>
