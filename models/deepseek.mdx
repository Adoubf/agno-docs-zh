DeepSeek 是一个提供大语言模型调用接口的平台。
可查阅其模型库[点击此处](https://api-docs.deepseek.com/quick_start/pricing)。

我们建议通过实验来寻找最适合您应用场景的模型。以下是通用推荐：
- `deepseek-chat` 模型适用于多数基础场景
- `deepseek-reasoner` 模型适合复杂推理和多步骤任务

DeepSeek 没有速率限制。高流量时段若遇响应延迟，请参阅其[文档](https://api-docs.deepseek.com/quick_start/rate_limit)了解处理方案。

## 认证

设置您的 `DEEPSEEK_API_KEY` 环境变量。密钥获取[点击这里](https://platform.deepseek.com/api_keys)。

<CodeGroup>

```bash Mac
export DEEPSEEK_API_KEY=***
```

```bash Windows
setx DEEPSEEK_API_KEY ***
```

</CodeGroup>

## 示例

使用 `DeepSeek` 搭配您的 `Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.deepseek import DeepSeek

agent = Agent(model=DeepSeek(), markdown=True)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>
