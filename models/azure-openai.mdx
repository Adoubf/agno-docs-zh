通过Azure基础设施使用OpenAI模型。了解更多请访问[这里](https://learn.microsoft.com/azure/ai-services/openai/overview)。

Azure OpenAI提供对`GPT-4o`、`o3-mini`等OpenAI模型的访问。

## 认证

访问[Azure门户](https://portal.azure.com/)中的Azure OpenAI并创建服务。然后使用Azure AI Studio门户创建部署并设置环境变量：

<CodeGroup>

```bash Mac
export AZURE_OPENAI_API_KEY=***
export AZURE_OPENAI_ENDPOINT=***  # Of the form https://<your-resource-name>.openai.azure.com/openai/deployments/<your-deployment-name>
# 可选：
# export AZURE_OPENAI_DEPLOYMENT=***
```

```bash Windows
setx AZURE_OPENAI_API_KEY ***  # Of the form https://<your-resource-name>.openai.azure.com/openai/deployments/<your-deployment-name>
setx AZURE_OPENAI_ENDPOINT ***
# 可选：
# 设置环境变量 AZURE_OPENAI_DEPLOYMENT 为 ***
```

</CodeGroup>

## 示例

使用`AzureOpenAI`与您的`Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.azure import AzureOpenAI
from os import getenv

agent = Agent(
    model=AzureOpenAI(id="gpt-4o"),
    markdown=True
)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

## 提示缓存

提示缓存将自动使用我们的`AzureOpenAI`模型完成。您可以在[OpenAI文档](https://platform.openai.com/docs/guides/prompt-caching)中阅读更多关于OpenAI如何处理缓存的信息。
