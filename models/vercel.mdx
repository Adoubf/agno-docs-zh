Vercel v0 API 提供大型语言模型服务，专为构建现代化网络应用而设计。它支持文本和图像输入，提供快速流式响应，并与 OpenAI 聊天补全 API 格式兼容。该接口针对前端和全栈网页开发的代码生成场景进行了优化。

更多详情请参阅 [Vercel v0 API 官方文档](https://vercel.com/docs/v0/api)。

## 认证

设置您的 `V0_API_KEY` 环境变量。您可以在 [v0.dev](https://v0.dev/chat/settings/keys) 创建 API 密钥。

<CodeGroup>

```bash Mac
export V0_API_KEY=your-v0-api-key
```

```bash Windows
setx V0_API_KEY your-v0-api-key
```

</CodeGroup>

## 示例

使用 `v0` 配合您的 `Agent`。以下示例假设您已将 `v0` Python 类（如您所提供）存放在 `agno/models/vercel.py` 路径。

<CodeGroup>

```python agent.py
from agno.agent import Agent
from agno.models.vercel import v0

agent = Agent(
    model=v0(id="v0-1.0-md"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Create a simple web app that displays a random number between 1 and 100.")

# 智能体打印响应("创建一个网页应用来获取城市天气并展示湿度、温度和风速卡片，使用shadcn组件和Tailwind CSS")

```

</CodeGroup>

<Note> 查看更多示例请访问 [这里](/examples/models/vercel)。 </Note>
