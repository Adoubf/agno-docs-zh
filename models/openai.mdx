GPT模型是业界领先的大型语言模型，被**智能体**用作默认的LLM。OpenAI提供多种世界级模型，具体可查看[模型文档](https://platform.openai.com/docs/models)。

建议通过实验找到最适合您用例的模型。以下是一些通用建议：

- `gpt-4o` 适用于大多数通用场景
- `gpt-4o-mini` 模型适合小型任务和快速推理
- `o1` 系列模型适合复杂推理和多步骤任务
- `o3-mini` 是具备工具调用和结构化输出能力的强大推理模型，且成本更低

OpenAI采用分级速率限制，详见[限制文档](https://platform.openai.com/docs/guides/rate-limits/usage-tiers)。

## 认证

设置您的`OPENAI_API_KEY`环境变量，可从[OpenAI获取API密钥](https://platform.openai.com/account/api-keys)。

<CodeGroup>

```bash Mac
export OPENAI_API_KEY=sk-***
```

```bash Windows
setx OPENAI_API_KEY sk-***
```

</CodeGroup>

## 示例

配合`Agent`使用`OpenAIChat`：

<CodeGroup>

```python agent.py

from agno.agent import Agent, RunResponse
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

## 提示缓存

提示缓存将自动通过我们的`OpenAIChat`模型实现。关于OpenAI的缓存机制，可查阅[其官方文档](https://platform.openai.com/docs/guides/prompt-caching)。

</CodeGroup>
