Hugging Face 提供了一系列最先进的语言模型，专为多样化NLP任务定制，
涵盖文本生成、摘要、翻译和问答等功能。
这些模型可通过Hugging Face Transformers库获取，
因其易用性、灵活性及完备的文档而被广泛采用。

探索HuggingFace语言模型[请点击此处](https://huggingface.co/docs/text-generation-inference/en/supported_models)。

## 认证

设置您的 `HF_TOKEN` 环境变量。访问HuggingFace[获取令牌](https://huggingface.co/settings/tokens)。

<CodeGroup>

```bash Mac
export HF_TOKEN=***
```

```bash Windows
setx HF_TOKEN ***
```

</CodeGroup>

## 示例

将 `HuggingFace` 与您的 `Agent` 配合使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.huggingface import HuggingFace

agent = Agent(
    model=HuggingFace(
        id="meta-llama/Meta-Llama-3-8B-Instruct",
        max_tokens=4096,
    ),
    markdown=True
)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> 查看更多示例[请访问此链接](../examples/models/huggingface)。 </Note>
