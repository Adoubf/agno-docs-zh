Meta提供了一套强大的多模态语言模型套件，这些模型以在广泛任务中展现卓越性能著称，包括出色的文本理解能力和视觉智能。

我们建议通过实验来寻找最适合您用例的模型。以下是一些通用推荐方案：

- `Llama-4-Scout-17B`：在包括多模态场景在内的大多数通用任务中表现优异
- `Llama-3.3-70B`：针对复杂推理任务的强大指令跟随模型

[在此](https://llama.developer.meta.com/docs/models)探索所有模型。

## 认证

设置您的 `LLAMA_API_KEY` 环境变量：

<CodeGroup>
```bash Mac
export LLAMA_API_KEY=YOUR_API_KEY
```
```bash Windows
setx LLAMA_API_KEY YOUR_API_KEY
```
</CodeGroup>

## 示例

使用 `Llama` 配合您的 `Agent`：

<CodeGroup>
```python agent.py
from agno.agent import Agent
from agno.models.meta import Llama

agent = Agent(
    model=Llama(
        id="Llama-4-Maverick-17B-128E-Instruct-FP8",
    ),
    markdown=True
)

agent.print_response("Share a 2 sentence horror story.")
```
</CodeGroup>
