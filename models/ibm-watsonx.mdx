IBM WatsonX 通过IBM云平台提供对强大基础模型的访问。

查看所有IBM WatsonX支持的模型[请点击这里](https://www.ibm.com/products/watsonx-ai/foundation-models)。

- 对于通用场景，我们推荐使用`meta-llama/llama-3-3-70b-instruct`
- 对于代码相关任务，我们推荐`ibm/granite-20b-code-instruct`
- 对于图像理解任务，我们推荐使用`meta-llama/llama-3-2-11b-vision-instruct`

#### 多模态支持

WatsonX支持将`Image`作为输入

## 身份验证

设置您的`IBM_WATSONX_API_KEY`和`IBM_WATSONX_PROJECT_ID`环境变量。请从[IBM Cloud](https://cloud.ibm.com/)获取凭证。
您也可以将`IBM_WATSONX_URL`环境变量设置为要使用的WatsonX API URL，默认值为`https://eu-de.ml.cloud.ibm.com`。

<CodeGroup>

```bash Mac（苹果电脑）
export IBM_WATSONX_API_KEY=***
export IBM_WATSONX_PROJECT_ID=***
```

```bash Windows
setx IBM_WATSONX_API_KEY ***
setx IBM_WATSONX_PROJECT_ID ***
```

</CodeGroup>

## 示例

使用`WatsonX`配合您的`Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.ibm import WatsonX

agent = Agent(
    model=WatsonX(id="meta-llama/llama-3-3-70b-instruct"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 查看更多示例[请访问](../examples/models/ibm)。</Note>
