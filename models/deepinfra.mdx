利用 DeepInfra 强大的指令模型及其他功能。

[DeepInfra](https://deepinfra.com) 支持多种模型。查看其模型库请访问[这里](https://deepinfra.com/models)。

我们建议通过实验找到最适合您用例的模型。以下是一些通用推荐：

- `deepseek-ai/DeepSeek-R1-Distill-Llama-70B` 模型擅长推理任务。
- `meta-llama/Llama-2-70b-chat-hf` 模型适合基础用例。
- `meta-llama/Llama-3.3-70B-Instruct` 模型适用于多步骤任务。

DeepInfra 存在速率限制。更多信息请查阅[文档](https://deepinfra.com/docs/advanced/rate-limits)。

## 认证

设置您的 `DEEPINFRA_API_KEY` 环境变量。获取密钥请访问[此处](https://deepinfra.com/dash/api_keys)。

<CodeGroup>

```bash Mac
export DEEPINFRA_API_KEY=***
```

```bash Windows
setx DEEPINFRA_API_KEY ***
```

</CodeGroup>

## 示例

将 `DeepInfra` 与您的 `Agent` 配合使用：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.deepinfra import DeepInfra

agent = Agent(
    model=DeepInfra(id="meta-llama/Llama-2-70b-chat-hf"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 查看更多示例请访问[这里](../examples/models/deepinfra)。 </Note>
