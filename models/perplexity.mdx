Perplexity 提供具备内置网络搜索功能的强大语言模型，可实现高级研究和问答功能。

探索 Perplexity 的模型[请点击此处](https://docs.perplexity.ai/guides/model-cards)。

## 认证

设置您的`PERPLEXITY_API_KEY`环境变量。获取密钥[请访问 Perplexity 这里](https://www.perplexity.ai/settings/api)。

<CodeGroup>

```bash Mac
export PERPLEXITY_API_KEY=***
```

```bash Windows
setx PERPLEXITY_API_KEY ***
```

</CodeGroup>

## 示例

使用`Perplexity`配合您的`Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.perplexity import Perplexity

agent = Agent(model=Perplexity(id="sonar-pro"), markdown=True)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story")

```

</CodeGroup>

<Note> 查看更多示例[请访问此处](../examples/models/perplexity)。</Note>
