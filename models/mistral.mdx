Mistral是一个为大语言模型提供端点服务的平台。  
查看其模型库[请点击此处](https://docs.mistral.ai/getting-started/models/models_overview/)。

我们建议通过实验来寻找最适合您使用场景的模型。以下是一些通用建议：

- `codestral` 模型擅长代码生成与编辑。
- `mistral-large-latest` 模型适用于大多数用例。
- `open-mistral-nemo` 是免费模型，适用于大多数用例。
- `pixtral-12b-2409` 是视觉模型，适用于OCR、文档转录和图像比对。其工具调用能力可能受限。

Mistral采用分级速率限制。详见[文档](https://docs.mistral.ai/deployment/laplateforme/tier/)。

## 身份验证

设置您的 `MISTRAL_API_KEY` 环境变量。密钥获取地址[在此](https://console.mistral.ai/api-keys/)。

<CodeGroup>

```bash Mac
export MISTRAL_API_KEY=***
```

```bash Windows
setx MISTRAL_API_KEY ***
```

</CodeGroup>

## 示例

配合您的 `Agent` 使用 `Mistral`：

<CodeGroup>

```python agent.py
import os

from agno.agent import Agent, RunResponse
from agno.models.mistral import MistralChat

mistral_api_key = os.getenv("MISTRAL_API_KEY")

agent = Agent(
    model=MistralChat(
        id="mistral-large-latest",
        api_key=mistral_api_key,
    ),
    markdown=True
)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 查看更多示例[请访问](../examples/models/mistral)。 </Note>
