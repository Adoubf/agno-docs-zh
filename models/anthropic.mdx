Claude是Anthropic开发的基础AI模型系列，适用于多种应用场景。
请参阅其[模型对比表](https://docs.anthropic.com/en/docs/about-claude/models#model-comparison-table)了解各模型差异。

我们建议通过实验选择最适合您用例的模型。以下是一些通用建议：
- `claude-3-5-sonnet-20241022` 模型适用于大多数场景并支持图像输入
- `claude-3-5-haiku-20241022` 模型是其速度最快的版本

Anthropic的API设有速率限制，详见官方[文档说明](https://docs.anthropic.com/en/api/rate-limits#response-headers)。

## 认证配置

设置您的 `ANTHROPIC_API_KEY` 环境变量。可通过[Anthropic控制台](https://console.anthropic.com/settings/keys)获取密钥。

<CodeGroup>

```bash Mac
export ANTHROPIC_API_KEY=***
```

```bash Windows
setx ANTHROPIC_API_KEY ***
```

</CodeGroup>

## 使用示例

使用 `Claude` 搭配您的 `Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(id="claude-3-5-sonnet-20240620"),
    markdown=True
)

# 在终端打印响应
agent.print_response("Share a 2 sentence horror story.")
```

## 提示缓存

通过设置 `cache_system_prompt` 为 `True`，可启用我们的 `Claude` 模型的系统提示缓存功能：

```python
from agno.agent import Agent
from agno.models.anthropic import Claude

agent = Agent(
    model=Claude(
        id="claude-3-5-sonnet-20241022",
        cache_system_prompt=True,
    ),
)
```

了解更多关于Agno `Claude` 模型的提示缓存技术，请参阅[此文档](https://docs.agno.com/examples/models/anthropic/prompt_caching)。

</CodeGroup>
