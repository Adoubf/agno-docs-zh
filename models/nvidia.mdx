NVIDIA提供了一系列针对高级NLP任务优化的高性能语言模型。这些模型属于NeMo框架的一部分，该框架提供高效训练、微调和部署最先进模型的工具。NVIDIA的语言模型专为处理大规模工作负载设计，通过GPU加速实现更快的推理和训练速度。我们建议尝试NVIDIA的模型，以找到最适合您应用的解决方案。

探索NVIDIA的模型[请点击这里](https://build.nvidia.com/models)。

## 认证

设置您的`NVIDIA_API_KEY`环境变量。从NVIDIA获取密钥[点击此处](https://build.nvidia.com/explore/discover)。

<CodeGroup>

```bash Mac
export NVIDIA_API_KEY=***
```

```bash Windows
setx NVIDIA_API_KEY ***
```

</CodeGroup>

## 示例

使用`Nvidia`配合您的`Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.nvidia import Nvidia

agent = Agent(model=Nvidia(), markdown=True)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story")

```

</CodeGroup>

<Note> 查看更多示例[请访问此处](../examples/models/nvidia)。 </Note>
