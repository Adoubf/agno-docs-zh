OpenRouter是一个为大型语言模型提供接入端点的平台。

## 认证

设置你的`OPENROUTER_API_KEY`环境变量。从[这里](https://openrouter.ai/settings/keys)获取密钥。

<CodeGroup>

```bash Mac
export OPENROUTER_API_KEY=***
```

```bash Windows
setx OPENROUTER_API_KEY ***
```

</CodeGroup>

## 示例

使用`OpenRouter`配合你的`Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.openrouter import OpenRouter

agent = Agent(
    model=OpenRouter(id="gpt-4o"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

## 提示缓存

当使用的服务提供商支持时，我们的`OpenRouter`模型会自动进行提示缓存。其他情况下，你可以通过`cache_control`请求头来激活该功能。
你可以在[OpenRouter文档](https://openrouter.ai/docs/features/prompt-caching)中阅读更多关于提示缓存的内容。
