title: Together

Together is a platform for providing endpoints for Large Language models.

## Authentication

Set your `TOGETHER_API_KEY` environment vairable.

<CodeGroup>

```bash Mac
export TOGETHER_API_KEY=***
```

```bash Windows
setx TOGETHER_API_KEY ***
```

</CodeGroup>

## Example

Use `Together` with your `Agent`:

<CodeGroup>

```python agent.py
from phi.agent import Agent, RunResponse
from phi.model.together import Together

agent = Agent(
    model=Together(id="meta-llama/Meta-Llama-3.1-8B-Instruct-Turbo"),
)

# Get the response in a variable
# run: RunResponse = agent.run("Share a 2 sentence horror story.")
# print(run.content)

# Print the response in the terminal
agent.print_response("Share a 2 sentence horror story.")


```

</CodeGroup>

## Params
<ResponseField name="id" type="str" default="mistralai/Mixtral-8x7B-Instruct-v0.1">
  The specific model ID used for generating responses.
</ResponseField>
<ResponseField name="name" type="str" default="Together">
  The name identifier for the Together agent.
</ResponseField>
<ResponseField name="provider" type="str">
  The provider of the model, combining "Together" with the model ID.
</ResponseField>
<ResponseField name="api_key" type="Optional[str]">
  The API key for authenticating requests to the Together service. Retrieved
  from the environment variable `TOGETHER_API_KEY`.
</ResponseField>
<ResponseField name="base_url" type="str" default="https://api.together.xyz/v1">
  The base URL for making API requests to the Together service.
</ResponseField>
<ResponseField name="monkey_patch" type="bool" default="False">
  Whether to apply monkey patching.
</ResponseField>
