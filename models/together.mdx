Together是一个为大型语言模型提供API端点的平台。
查看他们的模型库请点击[这里](https://www.together.ai/models)。

我们建议通过实验来寻找最适合您用例的模型。

Together采用分级速率限制策略。更多详情请参阅[文档](https://docs.together.ai/docs/rate-limits)。

## 身份验证

设置您的`TOGETHER_API_KEY`环境变量。获取API密钥请访问[Together平台](https://api.together.xyz/settings/api-keys)。

<CodeGroup>

```bash Mac（苹果电脑）
export TOGETHER_API_KEY=***
```

```bash Windows
setx TOGETHER_API_KEY ***
```

</CodeGroup>

## 示例

使用`Together`配合您的`Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.together import Together

agent = Agent(
    model=Together(id="meta-llama/Meta-Llama-3.1-8B-Instruct-Turbo"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> 查看更多示例请访问[这里](../examples/models/together)。 </Note>
