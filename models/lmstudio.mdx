使用 LM Studio 本地运行大型语言模型

[LM Studio](https://lmstudio.ai) 是一款出色的本地运行模型工具。

LM Studio 支持多种开源模型，模型库详见[此处](https://lmstudio.ai/models)。

我们建议通过实验找到最适合您用例的模型。以下是一些通用建议：

- `llama3.3` 模型适用于大多数基础用例。
- `qwen` 模型在工具使用场景中表现尤为出色。
- `deepseek-r1` 模型具备强大的推理能力。
- `phi4` 模型体积小巧但功能强大。

## 模型设置

安装 [LM Studio](https://lmstudio.ai)，下载所需模型并运行。

## 示例

在本地获取模型后，使用 `LM Studio` 模型类进行访问

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.lmstudio import LMStudio

agent = Agent(
    model=LMStudio(id="qwen2.5-7b-instruct-1m"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")
```

</CodeGroup>

<Note> 查看更多示例请访问[此处](../examples/models/lmstudio)。</Note>
