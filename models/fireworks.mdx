title: Fireworks
---

Fireworks is a platform for providing endpoints for Large Language models.

## Authentication

Set your `FIREWORKS_API_KEY` environment variable.

<CodeGroup>

```bash Mac
export FIREWORKS_API_KEY=***
```

```bash Windows
setx FIREWORKS_API_KEY ***
```

</CodeGroup>

## Example

Use `Fireworks` with your `Agent`:

<CodeGroup>

```python agent.py
from phi.agent import Agent, RunResponse
from phi.model.fireworks import Fireworks

agent = Agent(
    model=Fireworks(id="accounts/fireworks/models/firefunction-v2"),
)

# Get the response in a variable
# run: RunResponse = agent.run("Share a 2 sentence horror story.")
# print(run.content)

# Print the response in the terminal
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

## Params

<ResponseField name="name" type="str" default="Fireworks">
  The name identifier for the agent.
</ResponseField>
<ResponseField
  name="id"
  type="str"
  default="accounts/fireworks/models/firefunction-v2"
>
  The specific model ID used for generating responses.
</ResponseField>
<ResponseField name="api_key" type="str">
  The API key for authenticating requests to the service.
</ResponseField>
<ResponseField
  name="base_url"
  type="str"
  default="https://api.fireworks.ai/inference/v1"
>
  The base URL for making API requests to the service.
</ResponseField>
