Fireworks 是一个为大型语言模型提供端点的平台。

## 认证

设置您的 `FIREWORKS_API_KEY` 环境变量。从[此处](https://fireworks.ai/account/api-keys)获取密钥。

<CodeGroup>

```bash Mac
export FIREWORKS_API_KEY=***
```

```bash Windows
setx FIREWORKS_API_KEY ***
```

</CodeGroup>

## 提示缓存

提示缓存将自动使用我们的 `Fireworks` 模型完成。您可以在[Fireworks文档](https://docs.fireworks.ai/guides/prompt-caching#using-prompt-caching)中了解更多关于其缓存处理机制。

## 示例

使用 `Fireworks` 配合您的 `Agent`：

<CodeGroup>

```python agent.py
from agno.agent import Agent, RunResponse
from agno.models.fireworks import Fireworks

agent = Agent(
    model=Fireworks(id="accounts/fireworks/models/firefunction-v2"),
    markdown=True
)

# 在终端中打印响应
agent.print_response("Share a 2 sentence horror story.")

```

</CodeGroup>

<Note> 查看更多示例请访问[此处](../examples/models/fireworks)。 </Note>
