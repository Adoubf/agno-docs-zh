## OpenAI兼容集成方案

Cerebras也可通过OpenAI兼容接口使用，便于与需要OpenAI API的工具和库进行集成。

### 使用OpenAI兼容类

``CerebrasOpenAI``类为Cerebras模型提供了OpenAI风格的接口：

首先安装openai：

```shell
pip install openai
```

```python
from agno.agent import Agent
from agno.models.cerebras import CerebrasOpenAI

agent = Agent(
    model=CerebrasOpenAI(
        id="llama-4-scout-17b-16e-instruct",  # Model ID to use
        # base_url="https://api.cerebras.ai", # 可选：Cerebras的默认端点
    ),
    markdown=True,
)

# 在终端中打印响应
agent.print_response("write a two sentence horror story")
```

### 配置参数

``CerebrasOpenAI``类接收以下参数：

| 参数名       | 类型         | 描述                                                          | 默认值                        |
| ------------ | ------------ | ------------------------------------------------------------- | ----------------------------- |
| `id`        | str          | 模型标识符（如"llama-4-scout-17b-16e-instruct"）              | **必填**                      |
| `name`      | str          | 模型的显示名称                                                | "Cerebras"                    |
| `provider`  | str          | 供应商名称                                                    | "Cerebras"                    |
| `api_key`   | str          | API密钥（默认读取CEREBRAS_API_KEY环境变量）                   | None                          |
| `base_url`  | str          | Cerebras OpenAI兼容端点URL                                    | "https://api.cerebras.ai"     |
