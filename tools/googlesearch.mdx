title: Google Search
---

**GoogleSearch** enables an Agent to perform web crawling and scraping tasks.

## Example

The following agent will search Google for the latest news about "Mistral IA":

```python cookbook/tools/googlesearch_tools.py
from phi.agent import Agent
from phi.tools.googlesearch import GoogleSearch

news_agent = Agent(
    tools=[GoogleSearch()],
    description="You are a news agent that helps users find the latest news.",
    instructions=[
        "Given a topic by the user, respond with 4 latest news items about that topic.",
        "Search for 10 news items and select the top 4 unique items.",
        "Search in English and in French.",
    ],
    show_tool_calls=True,
    debug_mode=True,
)

news_agent.print_response("Mistral IA", markdown=True)
```

## Toolkit Params

<ResponseField name="fixed_max_results" type="int" default="None">
  Optional fixed maximum number of results to return. The default is None.
</ResponseField>
<ResponseField name="fixed_language" type="str" default="None">
  Optional fixed language for the requests. The default is None.
</ResponseField>
<ResponseField name="headers" type="Any" default="None">
  Optional headers to include in the requests. The default is None.
</ResponseField>
<ResponseField name="proxy" type="str" default="None">
  Optional proxy to be used for the requests. The default is None.
</ResponseField>
<ResponseField name="timeout" type="int" default="None">
  Optional timeout for the requests, in seconds. The default is None.
</ResponseField>

## Toolkit Functions

<ResponseField name="google_search">
  Searches Google for a specified query. Parameters include 'query' for the
  search term, 'max_results' for the maximum number of results (default is 5),
  and 'language' for the language of the search results (default is "en").
  Returns the search results as a JSON formatted string.
</ResponseField>
