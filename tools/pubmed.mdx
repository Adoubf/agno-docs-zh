**PubmedTools** enable an Agent to search for Pubmed for articles.

## Example

The following agent will search Pubmed for articles related to "ulcerative colitis".

```python cookbook/tools/pubmed.py
from phi.agent import Agent
from phi.tools.pubmed import PubmedTools

agent = Agent(tools=[PubmedTools()], show_tool_calls=True)
agent.print_response("Tell me about ulcerative colitis.")
```

## Toolkit Params

<ResponseField name="email" type="str" default="your_email@example.com">
  Specifies the email address to use.
</ResponseField>
<ResponseField name="max_results" type="int" default="None">
  Optional parameter to specify the maximum number of results to return. The
  default value is None.
</ResponseField>

## Toolkit Functions

<ResponseField name="search_pubmed">
  Searches PubMed for articles based on a specified query. Parameters include
  'query' for the search term and 'max_results' for the maximum number of
  results to return (default is 10). Returns a JSON string containing the search
  results, including publication date, title, and summary.
</ResponseField>
