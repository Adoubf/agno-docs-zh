title: Jina阅读器
---
**JinaReaderTools** 使智能体能够使用 Jina 执行网络搜索任务。

## 前提条件

以下示例需要 `jina` 库。

```shell
pip install -U jina
```

## 示例

以下智能体将使用 Jina API 对 https://github.com/AgnoAgi 的内容进行摘要。

```python cookbook/tools/jinareader.py
from agno.agent import Agent
from agno.tools.jina import JinaReaderTools

agent = Agent(tools=[JinaReaderTools()])
agent.print_response("Summarize: https://github.com/AgnoAgi")
```

## 工具包参数

| 参数            | 类型  | 默认值 | 描述                                                                 |
| -------------------- | ----- | ------- | -------------------------------------------------------------------------- |
| `api_key`            | `str` | -       | 用于身份验证的 API 密钥，从配置中获取。 |
| `base_url`           | `str` | -       | API 的基础 URL，从配置中获取。                 |
| `search_url`         | `str` | -       | 用于搜索查询的 URL，从配置中获取。         |
| `max_content_length` | `int` | -       | 允许的内容最大长度，从配置中获取。   |

## 工具包功能

| 功能       | 描述                                                                                                                                                                                            |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `read_url`     | 使用 Jina Reader API 读取指定 URL 的内容。参数包括 `url` 用于指定要读取的 URL。返回截断的内容或请求失败时的错误消息。                      |
| `search_query` | 使用 Jina Reader API 根据指定查询执行网络搜索。参数包括 `query` 用于搜索关键词。返回截断的搜索结果或请求失败时的错误消息。 |
