---
title: 网站工具
---
**网站工具**能让智能体解析网站内容并将其添加到知识库中。

## 前提条件

以下示例需要使用 `beautifulsoup4` 库。

```shell
pip install -U beautifulsoup4
```

## 示例

以下智能体会读取网站内容并将其添加到知识库。

```python cookbook/tools/website_tools.py
from agno.agent import Agent
from agno.tools.website import WebsiteTools

agent = Agent(tools=[WebsiteTools()], show_tool_calls=True)
agent.print_response("Search web page: 'https://docs.agno.com/introduction'", markdown=True)
```

## 工具包参数

| 参数        | 类型                   | 默认值 | 描述                                                                                                            |
| ---------------- | ---------------------- | ------- | ---------------------------------------------------------------------------------------------------------------------- |
| `knowledge_base` | `WebsiteKnowledgeBase` | -       | 与网站关联的知识库，包含与网站内容相关的各种数据和资源。 |

## 工具包函数

| 函数                        | 描述                                                                                                                                                                                                          |
| ------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `add_website_to_knowledge_base` | 该函数将网站内容添加到知识库。**注意：**网站必须以 `https://` 开头且应为有效网站。使用此函数可从互联网获取产品信息。 |
| `read_url`                      | 该函数读取URL并返回其内容。                                                                                                                                                                  |
