title: Crawl4AI
---
**Crawl4ai工具** 使智能体能够利用Crawl4ai库执行网络爬取和抓取任务。

## 前提条件

以下示例需要`crawl4ai`库。

```shell
pip install -U crawl4ai
```

## 示例

以下智能体将从https://github.com/agno-agi/agno网页抓取内容：

```python cookbook/tools/crawl4ai_tools.py
from agno.agent import Agent
from agno.tools.crawl4ai import Crawl4aiTools

agent = Agent(tools=[Crawl4aiTools(max_length=None)], show_tool_calls=True)
agent.print_response("Tell me about https://github.com/agno-agi/agno.")
```

## 工具包参数

| 参数         | 类型  | 默认值  | 描述                                                                       |
| ------------ | ----- | ------- | ------------------------------------------------------------------------- |
| `max_length` | `int` | `1000`  | 指定从网页返回的文本最大长度。                                             |

## 工具包函数

| 函数          | 描述                                                                                                                                                                                                 |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `web_crawler` | 使用crawl4ai的WebCrawler爬取网站。参数包括要爬取的'url'和可选的'max_length'（用于限制提取内容的长度）。'max_length'的默认值为1000。                                                                  |
