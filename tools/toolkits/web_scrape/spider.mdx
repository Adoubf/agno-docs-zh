**SpiderTools** 是一款开源网络爬虫工具，可直接返回适用于LLM的结构化数据。使用前需在 [Spider控制台](https://spider.cloud) 获取API密钥。

## 前提条件

以下示例需安装 `spider-client` 库。

```shell
pip install -U spider-client
```

## 示例

下列智能体会执行搜索查询获取美国最新新闻，并爬取第一条搜索结果。返回数据将以Markdown格式呈现。

```python cookbook/tools/spider_tools.py
from agno.agent import Agent
from agno.tools.spider import SpiderTools

agent = Agent(tools=[SpiderTools()])
agent.print_response('Can you scrape the first search result from a search on "news in USA"?', markdown=True)
```

## 工具包参数

| 参数名       | 类型   | 默认值 | 说明                             |
| ------------ | ------ | ------ | -------------------------------- |
| `max_results` | `int` | -      | 设定返回搜索结果的最大数量       |
| `url`         | `str` | -      | 指定待爬取或抓取的URL            |

## 工具包功能

| 函数       | 说明                     |
| ---------- | ------------------------ |
| `search` | 根据给定查询词执行网络搜索 |
| `scrape` | 爬取指定URL内容           |
| `crawl`  | 抓取指定URL内容           |
