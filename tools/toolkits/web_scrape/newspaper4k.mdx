**Newspaper4k** 使智能体能够通过 Newspaper4k 库阅读新闻文章。

## 前提条件

以下示例需要 `newspaper4k` 和 `lxml_html_clean` 库。

```shell
pip install -U newspaper4k lxml_html_clean
```

## 示例

以下智能体会总结这篇文章：https://www.rockymountaineer.com/blog/experience-icefields-parkway-scenic-drive-lifetime。

```python cookbook/tools/newspaper4k_tools.py
from agno.agent import Agent
from agno.tools.newspaper4k import Newspaper4kTools

agent = Agent(tools=[Newspaper4kTools()], debug_mode=True, show_tool_calls=True)
agent.print_response("Please summarize https://www.rockymountaineer.com/blog/experience-icefields-parkway-scenic-drive-lifetime")
```

## 工具包参数

| 参数               | 类型   | 默认值  | 描述                                                                 |
| ------------------ | ------ | ------- | -------------------------------------------------------------------- |
| `read_article`    | `bool` | `True`  | 启用读取文章全文内容的功能。                                         |
| `include_summary` | `bool` | `False` | 指定是否在返回全文内容的同时包含文章摘要。                           |
| `article_length`  | `int`  | -       | 设定待处理或返回的文章内容或其摘要的最大长度。                       |

## 工具包函数

| 函数               | 描述                                                   |
| ------------------ | ------------------------------------------------------ |
| `get_article_data` | 该函数读取文章的全文内容及相关数据。                   |
| `read_article`     | 该函数读取文章的全文内容。                             |
