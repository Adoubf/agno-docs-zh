**PandasTools** 使智能体能够使用 Pandas 库执行数据操作任务。

```python cookbook/tools/pandas_tool.py
from agno.agent import Agent
from agno.tools.pandas import PandasTools

# 创建带有 PandasTools 的智能体
agent = Agent(tools=[PandasTools()])

# 示例：创建包含样本数据的数据框并获取前5行
agent.print_response("""
Please perform these tasks:
1. Create a pandas dataframe named 'sales_data' using DataFrame() with this sample data:
   {'date': ['2023-01-01', '2023-01-02', '2023-01-03', '2023-01-04', '2023-01-05'],
    'product': ['Widget A', 'Widget B', 'Widget A', 'Widget C', 'Widget B'],
    'quantity': [10, 15, 8, 12, 20],
    'price': [9.99, 15.99, 9.99, 12.99, 15.99]}
2. Show me the first 5 rows of the sales_data dataframe
""")
```

## 工具包参数

| 参数                 | 类型                      | 默认值 | 描述                                                    |
| ------------------------- | ------------------------- | ------- | -------------------------------------------------------------- |
| `dataframes`              | `Dict[str, pd.DataFrame]` | `{}`    | 用于存储 Pandas 数据帧的字典，以其名称作为键。 |
| `create_pandas_dataframe` | `function`                | -       | 注册一个函数来创建 Pandas 数据帧。             |
| `run_dataframe_operation` | `function`                | -       | 注册一个函数在 Pandas 数据帧上运行操作。  |

## 工具包功能

| 函数                  | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `create_pandas_dataframe` | 使用指定的函数 `create_using_function` 和参数 `function_parameters` 创建一个名为 `dataframe_name` 的 Pandas 数据帧。参数包括 'dataframe_name'（数据帧名称）、'create_using_function'（用于创建数据帧的函数，例如 'read_csv'）和 'function_parameters'（函数所需的参数）。如果成功，则返回所创建数据帧的名称，否则返回错误消息。 |
| `run_dataframe_operation` | 在数据帧 `dataframe_name` 上运行指定的操作 `operation` 并使用参数 `operation_parameters`。参数包括 'dataframe_name'（要操作的数据帧）、'operation'（要执行的操作，例如 'head'、'tail'）和 'operation_parameters'（操作所需的参数）。如果成功，则返回操作结果，否则返回错误消息。                                          |
