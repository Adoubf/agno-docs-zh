**Discord工具** 使智能体能够在Discord中发送消息、读取消息历史、管理频道以及删除消息。

## 前提条件

以下示例需要从[此处](https://discord.com/developers/applications)获取Discord机器人令牌。

```shell
export DISCORD_BOT_TOKEN=***
```

## 示例

```python cookbook/tools/discord.py
from agno.agent import Agent
from agno.tools.discord import DiscordTools

agent = Agent(
    tools=[DiscordTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("Send 'Hello World!' to channel 1234567890", markdown=True)
```
## 工具包参数

| 参数                   | 类型    | 默认值 | 描述                                                                           |
| --------------------------- | ------- | ------- | ------------------------------------------------------------------------------------- |
| `bot_token`                 | `str`   | -  | 用于身份验证的Discord机器人令牌。                                                 |
| `enable_messaging`          | `bool`  | `True`  | 是否启用向频道发送消息功能。                                       |
| `enable_history`            | `bool`  | `True`  | 是否启用从频道检索消息历史功能。                           |
| `enable_channel_management` | `bool`  | `True`  | 是否启用获取频道信息及列出频道功能。                         |
| `enable_message_management` | `bool`  | `True`  | 是否启用从频道删除消息功能。                                    |

## 工具包函数

| 函数                                          | 描述                                                                                                   |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| `send_message`     | 向指定频道发送消息。返回成功或错误信息。                                    |
| `get_channel_info`               | 获取指定频道的信息。以JSON字符串形式返回频道信息。                    |
| `list_channels`                    | 列出指定服务器（公会）中的所有频道。以JSON形式返回频道列表。                        |
| `get_channel_messages`| 从指定频道检索消息历史。以JSON字符串形式返回消息。                         |
| `delete_message`| 通过ID从指定频道删除特定消息。返回成功或错误信息。                 |
