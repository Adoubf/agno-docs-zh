---
title: 维基百科
---
**Wikipedia工具集**使智能体能够搜索维基百科网站并将其内容添加到知识库中。

## 前提条件

以下示例需要 `wikipedia` 库。

```shell
pip install -U wikipedia
```

## 示例

以下智能体将搜索维基百科中的"ai"并打印响应。

```python cookbook/tools/wikipedia_tools.py
from agno.agent import Agent
from agno.tools.wikipedia import WikipediaTools

agent = Agent(tools=[WikipediaTools()], show_tool_calls=True)
agent.print_response("Search wikipedia for 'ai'")
```

## 工具包参数

| 名称             | 类型                     | 默认值 | 描述                                                                                                        |
| ---------------- | ------------------------ | ------- | ------------------------------------------------------------------------------------------------------------------ |
| `knowledge_base` | `WikipediaKnowledgeBase` | -       | 与维基百科关联的知识库，包含与维基百科内容相关的各种数据和资源。 |

## 工具包函数

| 函数名称                                | 描述                                                                                            |
| -------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| `search_wikipedia_and_update_knowledge_base` | 该函数搜索维基百科中的主题，将结果添加到知识库并返回。 |
| `search_wikipedia`                           | 在维基百科中搜索查询内容。                                                                        |
