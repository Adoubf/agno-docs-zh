**Pubmed工具**使智能体能够检索PubMed数据库中的文献。

## 示例

以下智能体将搜索PubMed中与"溃疡性结肠炎"相关的文献。

```python cookbook/tools/pubmed.py
from agno.agent import Agent
from agno.tools.pubmed import PubmedTools

agent = Agent(tools=[PubmedTools()], show_tool_calls=True)
agent.print_response("Tell me about ulcerative colitis.")
```

## 工具参数

| 参数         | 类型   | 默认值                   | 描述                                                                 |
| ------------ | ------ | ------------------------ | -------------------------------------------------------------------- |
| `email`       | `str` | `"your_email@example.com"` | 指定使用的电子邮箱地址。                                              |
| `max_results` | `int` | `None`                     | 可选参数，指定返回结果的最大数量。                                    |

## 工具函数

| 函数           | 描述                                                                                                                                                                                                                                                                                     |
| -------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `search_pubmed` | 根据指定查询条件搜索PubMed文献。参数包括用于搜索关键词的`query`和限制返回结果数量的`max_results`（默认值为10）。返回包含检索结果的JSON字符串，包括发表日期、标题和摘要信息。 |
