**HackerNews** 允许智能体搜索 Hacker News 网站。

## 示例

以下智能体会撰写一份引人入胜的摘要，包含 Hacker News 上热门的前 2 篇故事及其发布用户。

```python cookbook/tools/hackernews.py
from agno.agent import Agent
from agno.tools.hackernews import HackerNewsTools

agent = Agent(
    name="Hackernews Team",
    tools=[HackerNewsTools()],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response(
    "Write an engaging summary of the "
    "users with the top 2 stories on hackernews. "
    "Please mention the stories as well.",
)
```

## 工具包参数

| 参数          | 类型   | 默认值 | 描述                    |
| ------------------ | ------ | ------- | ------------------------------ |
| `get_top_stories`  | `bool` | `True`  | 启用获取热门故事功能。  |
| `get_user_details` | `bool` | `True`  | 启用获取用户详情功能。 |

## 工具包函数

| 函数                     | 描述                                                                                                                                                                      |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `get_top_hackernews_stories` | 从 Hacker News 获取热门故事。参数包含 `num_stories`（指定返回的故事数量，默认值为 10）。返回 JSON 格式的热门故事列表。 |
| `get_user_details`           | 通过用户名获取 Hacker News 用户的详细信息。参数包含 `username`（指定目标用户）。返回 JSON 格式的用户详情。                       |
