**DuckDuckGo** 允许智能体在互联网上搜索信息。

## 先决条件

以下示例需要 `duckduckgo-search` 库。要安装 DuckDuckGo，请运行以下命令：

```shell
pip install -U duckduckgo-search
```

## 示例

```python cookbook/tools/duckduckgo.py
from agno.agent import Agent
from agno.tools.duckduckgo import DuckDuckGoTools

agent = Agent(tools=[DuckDuckGoTools()], show_tool_calls=True)
agent.print_response("Whats happening in France?", markdown=True)
```

## 工具包参数

| 参数                | 类型    | 默认值  | 描述                                                                                          |
| ------------------- | ------- | ------- | ---------------------------------------------------------------------------------------------------- |
| `search`            | `bool` | `True`  | 启用 `duckduckgo_search` 函数以通过 DuckDuckGo 搜索查询。                |
| `news`              | `bool` | `True`  | 启用 `duckduckgo_news` 函数以通过 DuckDuckGo 获取最新新闻。           |
| `fixed_max_results` | `int`  | -       | 设置返回结果的最大数量上限。未提供默认值，使用时必须指定。 |
| `headers`           | `Any`  | -       | 接收任何类型的 HTTP 请求头信息。                                     |
| `proxy`             | `str`  | -       | 指定单个代理地址字符串用于 HTTP 请求。                       |
| `proxies`           | `Any`  | -       | 接收一个代理字典用于 HTTP 请求。                                        |
| `timeout`           | `int`  | `10`    | 设置 HTTP 请求超时时间（秒）。                                                      |

## 工具包函数

| 函数                | 描述                                               |
| ------------------- | --------------------------------------------------------- |
| `duckduckgo_search` | 使用此函数通过 DuckDuckGo 搜索查询。       |
| `duckduckgo_news`   | 使用此函数从 DuckDuckGo 获取最新新闻。 |
