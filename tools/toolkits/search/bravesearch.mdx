title: Brave 搜索
---
**BraveSearch** 使智能体能够使用 Brave 搜索引擎在互联网上搜索信息。

## 先决条件

以下示例需要 `brave-search` 库。

```shell
pip install -U brave-search
```
```shell
export BRAVE_API_KEY=***
```

## 示例

```python cookbook/tools/bravesearch_tools.py
from agno.agent import Agent
from agno.tools.bravesearch import BraveSearchTools

agent = Agent(
    tools=[BraveSearchTools()],
    description="You are a news agent that helps users find the latest news.",
    instructions=[
        "Given a topic by the user, respond with 4 latest news items about that topic."
    ],
    show_tool_calls=True,
)
agent.print_response("AI Agents", markdown=True)

```

## 工具包参数

| 参数           | 类型  | 默认值 | 说明                                         |
| ------------------- | ----- | ------- | --------------------------------------------------- |
| `fixed_max_results` | `int` | `None`  | 可选固定返回的最大结果数。 |
| `fixed_language`    | `str` | `None`  | 可选固定请求语言。           |


## 工具包函数

| 函数        | 说明                                                                                                                                                                                                                                                                            |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `brave_search` | 在 Brave 中搜索指定查询。参数包括 `query`（搜索关键词）、`max_results`（最大结果数，默认为 5）、`country`（搜索结果的地理区域，默认为 "US"）以及 `language`（搜索结果的语言，默认为 "en"）。返回的搜索结果以 JSON 格式化字符串呈现。 |
