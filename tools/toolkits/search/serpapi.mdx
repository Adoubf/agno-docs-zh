**SerpApi工具**能让智能体在Google和YouTube上执行搜索查询。

## 前提条件

以下示例需要`google-search-results`库以及来自[SerpApi](https://serpapi.com/)的API密钥。

```shell
pip install -U google-search-results
```

```shell
export SERP_API_KEY=***
```

## 示例

以下智能体会在Google上搜索查询："Whats happening in the USA"并分享结果。

```python cookbook/tools/serpapi_tools.py
from agno.agent import Agent
from agno.tools.serpapi import SerpApiTools

agent = Agent(tools=[SerpApiTools()])
agent.print_response("Whats happening in the USA?", markdown=True)
```

## 工具包参数

| 参数        | 类型   | 默认值 | 描述                                                 |
| ---------------- | ------ | ------- | ----------------------------------------------------------- |
| `api_key`        | `str`  | -       | 用于身份验证的API密钥。                        |
| `search_youtube` | `bool` | `False` | 启用YouTube内容搜索功能。 |

## 工具包函数

| 函数         | 描述                                |
| ---------------- | ------------------------------------------ |
| `search_google`  | 该函数在Google上执行查询搜索。 |
| `search_youtube` | 在YouTube上执行查询搜索。              |
