## 示例

**Searxng** 使智能体能够针对查询进行网络搜索、抓取网站或爬取网站内容。

```python cookbook/tools/searxng_tools.py
from agno.agent import Agent
from agno.tools.searxng import SearxngTools

# 初始化 Searxng 并配置您的 Searxng 实例 URL
searxng = SearxngTools(
    host="http://localhost:53153",
    engines=[],
    fixed_max_results=5,
    news=True,
    science=True
)

# 创建一个带有Searxng的智能体
agent = Agent(tools=[searxng])

# 示例：让智能体使用 Searxng 进行搜索
agent.print_response("""
Please search for information about artificial intelligence
and summarize the key points from the top results
""")
```

## 工具包参数

| 参数                | 类型         | 默认值   | 描述                                                        |
| ------------------- | ----------- | ------- | ------------------------------------------------------------------ |
| `host`              | `str`       | -       | 连接所用的主机地址。                                       |
| `engines`           | `List[str]` | `[]`    | 使用的搜索引擎列表。                                   |
| `fixed_max_results` | `int`       | `None`  | 可选参数，用于指定返回结果的固定最大数量。 |
| `images`            | `bool`      | `False` | 启用图片搜索功能。                                      |
| `it`                | `bool`      | `False` | 启用IT相关内容搜索功能。                          |
| `map`               | `bool`      | `False` | 启用地图搜索功能。                                        |
| `music`             | `bool`      | `False` | 启用音乐搜索功能。                                       |
| `news`              | `bool`      | `False` | 启用新闻搜索功能。                                        |
| `science`           | `bool`      | `False` | 启用科学相关内容搜索功能。                     |
| `videos`            | `bool`      | `False` | 启用视频搜索功能。                                      |

## 工具包函数

| 函数                | 描述                                                                                                                                                                                                                         |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `search`         | 使用指定查询执行通用网络搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回搜索结果。                             |
| `image_search`   | 使用指定查询执行图片搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回图片搜索结果。                            |
| `it_search`      | 使用指定查询执行IT相关信息搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回IT相关搜索结果。   |
| `map_search`     | 使用指定查询执行地图搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回地图搜索结果。                            |
| `music_search`   | 使用指定查询执行音乐相关信息搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回音乐搜索结果。     |
| `news_search`    | 使用指定查询执行新闻搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回新闻搜索结果。                           |
| `science_search` | 使用指定查询执行科学相关信息搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回科学搜索结果。 |
| `video_search`   | 使用指定查询执行视频搜索。参数包括搜索关键词 `query` 和返回结果最大数量 `max_results`（默认为5）。返回视频搜索结果。                        |
