**GoogleSearch** 功能使智能体能够执行网络爬取与抓取任务。

## 前提条件

以下示例需要 `googlesearch` 和 `pycountry` 库支持。

```shell
pip install -U googlesearch-python pycountry
```

## 示例

以下智能体会在Google上搜索关于"Mistral AI"的最新消息：

```python cookbook/tools/googlesearch_tools.py
from agno.agent import Agent
from agno.tools.googlesearch import GoogleSearchTools

agent = Agent(
    tools=[GoogleSearchTools()],
    description="You are a news agent that helps users find the latest news.",
    instructions=[
        "Given a topic by the user, respond with 4 latest news items about that topic.",
        "Search for 10 news items and select the top 4 unique items.",
        "Search in English and in French.",
    ],
    show_tool_calls=True,
    debug_mode=True,
)

agent.print_response("Mistral AI", markdown=True)
```

## 工具包参数

| 参数名           | 类型  | 默认值 | 描述                                         |
| ------------------- | ----- | ------- | --------------------------------------------------- |
| `fixed_max_results` | `int` | `None`  | 可选的固定返回结果最大数量。 |
| `fixed_language`    | `str` | `None`  | 可选的请求固定语言设置。           |
| `headers`           | `Any` | `None`  | 可选的要包含在请求中的头信息。        |
| `proxy`             | `str` | `None`  | 可选的要用于请求的代理服务器。         |
| `timeout`           | `int` | `None`  | 可选的请求超时时间（单位：秒）。      |

## 工具包函数

| 函数名        | 描述                                                                                                                                                                                                                                                                            |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `google_search` | 在Google搜索指定查询。参数包括 `query`（搜索关键词），`max_results`（返回结果最大数量，默认5），以及 `language`（搜索结果语言，默认"en"）。以JSON格式字符串返回搜索结果。 |
