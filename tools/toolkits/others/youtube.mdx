**YouTube工具**能让智能体在获取YouTube视频URL后，访问其字幕和元数据。

## 前提条件

以下示例需要使用`youtube_transcript_api`库。

```shell
pip install -U youtube_transcript_api
```

## 示例

以下智能体会提供YouTube视频的摘要。

```python cookbook/tools/youtube_tools.py
from agno.agent import Agent
from agno.tools.youtube import YouTubeTools

agent = Agent(
    tools=[YouTubeTools()],
    show_tool_calls=True,
    description="You are a YouTube agent. Obtain the captions of a YouTube video and answer questions.",
)

agent.print_response("Summarize this video https://www.youtube.com/watch?v=Iv9dewmcFbs&t", markdown=True)
```

## 工具包参数

| 参数                | 类型        | 默认值  | 描述                                                                        |
| -------------------- | ----------- | ------- | ---------------------------------------------------------------------------------- |
| `get_video_captions` | `bool`      | `True`  | 启用获取视频字幕的功能。                              |
| `get_video_data`     | `bool`      | `True`  | 启用获取视频元数据及其他相关数据的功能。       |
| `languages`          | `List[str]` | -       | 指定应获取数据的语言列表（如适用）。 |

## 工具包函数

| 函数                     | 描述                                              |
| ---------------------------- | -------------------------------------------------------- |
| `get_youtube_video_captions` | 该函数获取YouTube视频的字幕。 |
| `get_youtube_video_data`     | 该函数获取YouTube视频的元数据。 |
