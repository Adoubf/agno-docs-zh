**YFinanceTools** 使智能体能够从雅虎财经获取股票数据、财务信息等内容。

## 前提条件

以下示例需要 `yfinance` 库。

```shell
pip install -U yfinance
```

## 示例

以下智能体将提供关于NVDA（英伟达公司）的股价和分析师推荐信息。

```python cookbook/tools/yfinance_tools.py
from agno.agent import Agent
from agno.tools.yfinance import YFinanceTools

agent = Agent(
    tools=[YFinanceTools(stock_price=True, analyst_recommendations=True, stock_fundamentals=True)],
    show_tool_calls=True,
    description="You are an investment analyst that researches stock prices, analyst recommendations, and stock fundamentals.",
    instructions=["Format your response using markdown and use tables to display data where possible."],
)
agent.print_response("Share the NVDA stock price and analyst recommendations", markdown=True)
```

## 工具包参数

| 参数                 | 类型 | 默认值 | 描述                                                                    |
| ------------------------- | ---- | ------- | ------------------------------------------------------------------------------ |
| `stock_price`             | `bool` | `True`  | 启用获取当前股票价格信息的功能。         |
| `company_info`            | `bool` | `False` | 启用获取详细公司信息的功能。            |
| `stock_fundamentals`      | `bool` | `False` | 启用获取股票基本面数据的功能。          |
| `income_statements`       | `bool` | `False` | 启用获取公司收入报表的功能。          |
| `key_financial_ratios`    | `bool` | `False` | 启用获取公司关键财务比率的功能。      |
| `analyst_recommendations` | `bool` | `False` | 启用获取股票分析师推荐信息的功能。     |
| `company_news`            | `bool` | `False` | 启用获取公司相关最新新闻的功能。    |
| `technical_indicators`    | `bool` | `False` | 启用获取股票分析技术指标的功能。 |
| `historical_prices`       | `bool` | `False` | 启用获取股票历史价格数据的功能。       |

## 工具包函数

| 函数                      | 描述                                                      |
| ----------------------------- | ---------------------------------------------------------------- |
| `get_current_stock_price`     | 该函数获取公司的当前股票价格。    |
| `get_company_info`            | 该函数获取公司的详细信息。    |
| `get_historical_stock_prices` | 该函数获取公司的历史股票价格。   |
| `get_stock_fundamentals`      | 该函数获取股票的基本面数据。          |
| `get_income_statements`       | 该函数获取公司的收入报表。          |
| `get_key_financial_ratios`    | 该函数获取公司的关键财务比率。      |
| `get_analyst_recommendations` | 该函数获取股票的分析师推荐信息。     |
| `get_company_news`            | 该函数获取公司相关的最新新闻。    |
| `get_technical_indicators`    | 该函数获取股票分析的技术指标。 |
