**OpenBBTools** 使智能体能够提供股票和公司相关信息。

```python cookbook/tools/openbb_tools.py
from agno.agent import Agent
from agno.tools.openbb import OpenBBTools


agent = Agent(tools=[OpenBBTools()], debug_mode=True, show_tool_calls=True)

# 展示股票分析的示例用法
agent.print_response(
    "Get me the current stock price and key information for Apple (AAPL)"
)

# 市场分析示例
agent.print_response(
    "What are the top gainers in the market today?"
)

# 展示经济指标的示例
agent.print_response(
    "Show me the latest GDP growth rate and inflation numbers for the US"
)
```

## 工具包参数

| 参数          | 类型   | 默认值 | 描述                                                                        |
| ------------- | ------ | ------ | -------------------------------------------------------------------------- |
| `read_article` | `bool` | `True` | 启用读取文章完整内容的功能。                                                 |
| `include_summary` | `bool` | `False` | 指定是否在返回结果中包含文章的摘要信息。                                      |
| `article_length` | `int` | -       | 设定待处理或返回的文章内容/摘要的最大长度限制。                               |

## 工具包功能

| 函数                | 描述                                                                       |
| ------------------- | ------------------------------------------------------------------------- |
| `get_stock_price`   | 获取单个或多个股票代码的实时股价。                                          |
| `search_company_symbol` | 根据公司名称搜索对应的股票代码。                                            |
| `get_price_targets` | 获取单个或多个股票代码的价格目标数据。                                      |
| `get_company_news`  | 获取单个或多个股票代码的最新新闻资讯。                                      |
| `get_company_profile` | 获取单个或多个股票代码对应的公司概况信息。                                  |
