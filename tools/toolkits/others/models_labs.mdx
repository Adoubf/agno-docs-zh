## 先决条件

您需要安装 `requests` 库。

```bash
pip install requests
```

设置 `MODELS_LAB_API_KEY` 环境变量。

```bash
export MODELS_LAB_API_KEY=****
```

## 示例

以下智能体会使用 ModelsLabs 根据文本提示生成视频。

```python cookbook/tools/models_labs_tools.py
from agno.agent import Agent
from agno.tools.models_labs import ModelsLabsTools

# 使用 ModelsLabs 工具创建智能体
agent = Agent(tools=[ModelsLabsTools()], name="ModelsLabs Agent")

agent.print_response("Generate a video of a beautiful sunset over the ocean", markdown=True)
```

## 工具包参数

| 参数             | 类型   | 默认值                                           | 描述                                                                 |
| --------------------- | ------ | ------------------------------------------------- | -------------------------------------------------------------------------- |
| `api_key`       | `str`  | `None`                                 | 用于身份验证的 ModelsLab API 密钥                                   |
| `wait_for_completion`       | `bool`  | `False`                                | 是否等待视频准备完成                                  |
| `add_to_eta`      | `int` | `15`                                | 为预估时间添加的额外缓冲时间，以补偿获取视频所需时间 |
| `max_wait_time`      | `int` | `60`                                | 等待视频准备完成的最大时间                             |
| `file_type`      | `str` | `"mp4"`                                | 生成的文件类型                                               |

## 工具包功能

| 函数         | 描述                                     |
| ---------------- | ----------------------------------------------- |
| `generate_media` | 根据文本提示生成视频或 GIF 动画 |
