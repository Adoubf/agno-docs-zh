**ElevenLabsTools** 使智能体能够使用[ElevenLabs](https://elevenlabs.io/docs/product/introduction)执行音频生成任务

## 前提条件

您需要安装 `elevenlabs` 库并获取可从[Eleven Labs](https://elevenlabs.io/)获得的API密钥

```bash
pip install elevenlabs
```

设置 `ELEVEN_LABS_API_KEY` 环境变量。

```bash
export ELEVEN_LABS_API_KEY=****
```

## 示例

以下智能体将使用 Eleven Labs 根据用户提示生成音频。

```python cookbook/tools/eleven_labs_tools.py
from agno.agent import Agent
from agno.tools.eleven_labs import ElevenLabsTools

# 使用 ElevenLabs 工具创建智能体
agent = Agent(tools=[
    ElevenLabsTools(
        voice_id="JBFqnCBsd6RMkjVDRZzb", model_id="eleven_multilingual_v2", target_directory="audio_generations"
    )
], name="ElevenLabs Agent")

agent.print_response("Generate a audio summary of the big bang theory", markdown=True)
```

## 工具包参数

| 参数          | 类型            | 默认值                  | 描述                                                                                                                                                                    |
| ------------------ | --------------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `api_key`          | `str`           | `None`                   | 用于认证的 Eleven Labs API 密钥                                                                                                                                     |
| `voice_id`         | `str`           | `JBFqnCBsd6RMkjVDRZzb`   | 用于音频生成的语音 ID                                                                                                                                   |
| `target_directory` | `Optional[str]` | `None`                   | 保存音频文件的目录                                                                                                                                           |
| `model_id`         | `str`           | `eleven_multilingual_v2` | 用于音频生成的模型 ID                                                                                                                                 |
| `output_format`    | `str`           | `mp3_44100_64`           | 音频生成的输出格式（更多信息请查看[文档](https://elevenlabs.io/docs/api-reference/text-to-speech#parameter-output-format)） |

## 工具包功能

| 函数                | 描述                                     |
| ----------------------- | ----------------------------------------------- |
| `text_to_speech`        | 将文本转换为语音                          |
| `generate_sound_effect` | 根据文本提示生成音效音频 |
| `get_voices`            | 获取可用的语音列表                |
