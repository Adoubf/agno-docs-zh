---
**AgentQLTools** 使智能体能够通过 AgentQL API 浏览和抓取网站内容。

## 前提条件

以下示例需要 `agentql` 库和一个从 [AgentQL](https://agentql.com/) 获取的 API 令牌。

```shell
pip install -U agentql
```

```shell
export AGENTQL_API_KEY=***
```

## 示例

以下智能体会打开浏览器并抓取页面上的所有文本内容。

```python cookbook/tools/agentql_tools.py
from agno.agent import Agent
from agno.models.openai import OpenAIChat
from agno.tools.agentql import AgentQLTools

agent = Agent(
    model=OpenAIChat(id="gpt-4o"), tools=[AgentQLTools()], show_tool_calls=True
)

agent.print_response("https://docs.agno.com/introduction", markdown=True)
```

<Note>
  AgentQL 将启动一个浏览器实例（请勿关闭）并在网站上进行数据抓取。
</Note>

## 工具包参数

| 参数           | 类型   | 默认值  | 描述                         |
| -------------- | ------ | ------- | ---------------------------- |
| `api_key`       | `str`  | `None`  | AgentQL 的 API 密钥          |
| `scrape`        | `bool` | `True`  | 是否使用文本抓取工具         |
| `agentql_query` | `str`  | `None`  | 自定义 AgentQL 查询语句      |

## 工具包功能

| 函数                | 描述                                          |
| ------------------- | -------------------------------------------- |
| `scrape_website`        | 用于抓取网页上的所有文本内容                  |
| `custom_scrape_website` | 使用自定义的 `agentql_query` 来抓取网页内容  |
