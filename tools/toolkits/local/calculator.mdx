title: 计算器
---
**计算器**功能使智能体能够执行数学运算。

## 示例

以下智能体会计算`10*5`的结果，然后将其提升至`2`次方：

```python cookbook/tools/calculator_tools.py
from agno.agent import Agent
from agno.tools.calculator import CalculatorTools

agent = Agent(
    tools=[
        CalculatorTools(
            add=True,
            subtract=True,
            multiply=True,
            divide=True,
            exponentiate=True,
            factorial=True,
            is_prime=True,
            square_root=True,
        )
    ],
    show_tool_calls=True,
    markdown=True,
)

agent.print_response("What is 10*5 then to the power of 2, do it step by step")
```

## 工具包参数

| 参数名      | 类型   | 默认值 | 说明                                                         |
| -------------- | ------ | ------- | ------------------------------------------------------------------- |
| `add`          | `bool` | `True`  | 启用加法运算功能                      |
| `subtract`     | `bool` | `True`  | 启用减法运算功能                   |
| `multiply`     | `bool` | `True`  | 启用乘法运算功能                |
| `divide`       | `bool` | `True`  | 启用除法运算功能                      |
| `exponentiate` | `bool` | `False` | 启用幂运算功能                |
| `factorial`    | `bool` | `False` | 启用计算数字阶乘的功能   |
| `is_prime`     | `bool` | `False` | 启用检查数字是否为质数的功能            |
| `square_root`  | `bool` | `False` | 启用计算数字平方根的功能 |

## 工具包函数

| 函数名       | 说明                                                                              |
| -------------- | ---------------------------------------------------------------------------------------- |
| `add`          | 将两个数字相加并返回结果                                                 |
| `subtract`     | 用第一个数字减去第二个数字并返回结果                       |
| `multiply`     | 将两个数字相乘并返回结果                                           |
| `divide`       | 用第一个数字除以第二个数字并返回结果。处理除以零的情况 |
| `exponentiate` | 将第一个数字提升至第二个数字的幂次并返回结果        |
| `factorial`    | 计算数字的阶乘并返回结果。处理负数情况   |
| `is_prime`     | 检查数字是否为质数并返回结果                                      |
| `square_root`  | 计算数字的平方根并返回结果。处理负数情况 |
