## Toolkit Params

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | `str` | `"sleep"` | The name of the tool |

## Toolkit Functions

| Function | Description |
|----------|-------------|
| `sleep` | Pauses execution for a specified number of seconds |
