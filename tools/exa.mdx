**ExaTools** enable an Agent to search the web using Exa.

## Example

The following agent will run seach exa for AAPL news and print the response.

```python cookbook/tools/exa_tools.py
import os

from phi.agent import Agent
from phi.tools.exa import ExaTools

os.environ["EXA_API_KEY"] = "your api key"

assistant = Agent(
    tools=[ExaTools(include_domains=["cnbc.com", "reuters.com", "bloomberg.com"])], show_tool_calls=True
)
assistant.print_response("Search for AAPL news", debug_mode=True, markdown=True)
```

## Toolkit Params

<ResponseField name="api_key" type="str">
  API key for authentication purposes.
</ResponseField>
<ResponseField name="search" type="bool" default="False">
  Determines whether to enable search functionality.
</ResponseField>
<ResponseField name="search_with_contents" type="bool" default="True">
  Indicates whether to include contents in the search results.
</ResponseField>
<ResponseField name="show_results" type="bool" default="False">
  Controls whether to display search results directly.
</ResponseField>

## Toolkit Functions

<ResponseField name="search_exa">Searches Exa for a query.</ResponseField>
<ResponseField name="search_exa_with_contents">
  Searches Exa for a query and returns the contents from the search results.
</ResponseField>
