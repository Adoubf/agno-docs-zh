title: Mistral
---

Mistral is a platform for providing endpoints for Large Language models.

## Authentication

Set your `MISTRAL_API_KEY` environment variable.

<CodeGroup>

```bash Mac
export MISTRAL_API_KEY=***
```

```bash Windows
setx MISTRAL_API_KEY ***
```

</CodeGroup>

## Usage

Use `Mistral` with your `Assistant`:

<CodeGroup>

```python assistant.py
from phi.assistant import Assistant
from phi.llm.Mistral import Mistral

assistant = Assistant(
    llm=Mistral(model="mistral-large-latest"),
    description="You help people with their health and fitness goals.",
)
assistant.print_response("Share a quick healthy breakfast recipe.", markdown=True)
```

</CodeGroup>

## Params

<ResponseField name="name" type="str" default="Mistral">
  The name identifier for the assistant.
</ResponseField>
<ResponseField name="model" type="str" default="mistral-large-latest">
  The specific model ID used for generating responses.
</ResponseField>
<ResponseField name="temperature" type="float">
  The sampling temperature to use, between 0 and 2. Higher values like 0.8 will
  make the output more random, while lower values like 0.2 will make it more
  focused and deterministic.
</ResponseField>
<ResponseField name="max_tokens" type="int" default="1024">
  The maximum number of tokens to generate in the response.
</ResponseField>
<ResponseField name="top_p" type="float">
  The nucleus sampling parameter. The model considers the results of the tokens
  with top_p probability mass.
</ResponseField>
<ResponseField name="random_seed" type="int">
  The seed for random number generation to ensure reproducibility of results.
</ResponseField>
<ResponseField name="safe_mode" type="bool">
  Enable safe mode to filter potentially harmful or inappropriate content.
</ResponseField>
<ResponseField name="safe_prompt" type="bool">
  Enable safe prompt mode to filter potentially harmful or inappropriate
  prompts.
</ResponseField>
<ResponseField
  name="response_format"
  type="Union[Dict[str, Any], ChatCompletionResponseFormat]"
>
  The format of the response, either as a dictionary or as a
  ChatCompletionResponseFormat object.
</ResponseField>
<ResponseField name="api_key" type="str">
  The API key for authenticating requests to the service.
</ResponseField>
<ResponseField name="endpoint" type="str">
  The API endpoint URL for making requests to the service.
</ResponseField>
<ResponseField name="max_retries" type="int">
  The maximum number of retry attempts for failed requests.
</ResponseField>
<ResponseField name="timeout" type="int">
  The timeout duration for requests, specified in seconds.
</ResponseField>
<ResponseField name="mistral_client" type="MistralClient">
  An instance of MistralClient provided for making API requests.
</ResponseField>
