title: Fireworks
---

Fireworks is a platform for providing endpoints for Large Language models.

## Authentication

Set your `FIREWORKS_API_KEY` environment variable.

<CodeGroup>

```bash Mac
export FIREWORKS_API_KEY=***
```

```bash Windows
setx FIREWORKS_API_KEY ***
```

</CodeGroup>

## Example

Use `Fireworks` with your `Agent`:

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.llm.fireworks import Fireworks

agent = Agent(
    llm=Fireworks(model="accounts/fireworks/models/firefunction-v1"),
    description="You help people with their health and fitness goals.",
)
agent.print_response("Share a quick healthy breakfast recipe.", markdown=True)
```

</CodeGroup>

## Params

<ResponseField name="name" type="str" default="Fireworks">
  The name identifier for the agent.
</ResponseField>
<ResponseField
  name="model"
  type="str"
  default="accounts/fireworks/models/firefunction-v1"
>
  The specific model ID used for generating responses.
</ResponseField>
<ResponseField name="api_key" type="str">
  The API key for authenticating requests to the service.
</ResponseField>
<ResponseField
  name="base_url"
  type="str"
  default="https://api.fireworks.ai/inference/v1"
>
  The base URL for making API requests to the service.
</ResponseField>
