title: OpenRouter
---

OpenRouter is a platform for providing endpoints for Large Language models.

## Authentication

Set your `OPENROUTER_API_KEY` environment variable.

<CodeGroup>

```bash Mac
export OPENROUTER_API_KEY=***
```

```bash Windows
setx OPENROUTER_API_KEY ***
```

</CodeGroup>

## Usage

Use `OpenRouter` with your `Assistant`:

<CodeGroup>

```python assistant.py
from phi.assistant import Assistant
from phi.llm.openrouter import OpenRouter

assistant = Assistant(
    llm=OpenRouter(model="mistralai/mistral-7b-instruct:free"),
    description="You help people with their health and fitness goals.",
)
assistant.print_response("Share a 2 sentence quick and healthy breakfast recipe.", markdown=True)
```

</CodeGroup>

## Params

<ResponseField name="name" type="str" default="OpenRouter">
  The name identifier for the OpenRouter assistant.
</ResponseField>
<ResponseField
  name="model"
  type="str"
  default="mistralai/mistral-7b-instruct:free"
>
  The specific model ID used for generating responses.
</ResponseField>
<ResponseField name="api_key" type="Optional[str]">
  The API key for authenticating requests to the OpenRouter service. Retrieved
  from the environment variable `OPENROUTER_API_KEY`.
</ResponseField>
<ResponseField
  name="base_url"
  type="str"
  default="https://openrouter.ai/api/v1"
>
  The base URL for making API requests to the OpenRouter service.
</ResponseField>
