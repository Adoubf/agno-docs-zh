The [AI Cookbook](https://github.com/phidatahq/ai-cookbook) contains a bunch of recipes for building different types of AI products. From simple conversations to advanced fine-tuning and evaluations.

You can run each recipe individually or clone the [AI Cookbook](https://github.com/phidatahq/ai-cookbook) repo and run it from there.

## Recipes

- [Simple Conversation](/ai-cookbook/simple-conversation)
- [Rag Conversation](/ai-cookbook/rag-conversation)
- [Autonomous Conversation](/ai-cookbook/auto-conversation)
- [Conversation with Storage](/ai-cookbook/conversation-w-storage)
- [Function Calling: Trending On Hackernews](/ai-cookbook/trending-on-hn)
- [Getting Pydantic outputs from LLMs: Movie Generator](/ai-cookbook/movie-generator)
- [Chaining multiple tasks: Short Story Generator](/ai-cookbook/short-story-generator)
- [Using LangChain Retriever as a knowledge base](/ai-cookbook/langchain-retriever)

## Setup

<Steps>
  <Step title="Clone the AI Cookbook repo">
    clone the [ai-cookbook](https://github.com/phidatahq/ai-cookbook) repo

    ```bash
    git clone https://github.com/phidatahq/ai-cookbook.git
    ```

    cd into the `ai-cookbook` directory

    ```bash
    cd ai-cookbook
    ```

  </Step>
  <Step title="Create a virtual environment">
    Create a virtual environment with the required libraries and install the project in editable mode. You can use a helper script or run these steps manually.

    <CodeGroup>

    ```bash Mac
    ./scripts/create_venv.sh
    source aienv/bin/activate
    ```

    ```bash Manual (Mac)
    python3 -m venv aienv
    source aienv/bin/activate

    pip install -r requirements.txt
    pip install --editable .
    ```

    ```bash Windows
    python3 -m venv aienv
    aienv/scripts/activate

    pip install -r requirements.txt
    pip install --editable .
    ```
    </CodeGroup>

  </Step>
  <Step title="Install docker">
    Install [docker desktop](https://docs.docker.com/desktop/install/mac-install/) to run the cookbook resources locally
  </Step>
  <Step title="Set your OpenAI key">

    Set the `OPENAI_API_KEY` environment variable to your OpenAI Key. You can get one [from OpenAI here](https://platform.openai.com/account/api-keys). Otherwise `phi` provides ready to use access to OpenAI models.

    <CodeGroup>

    ```bash Mac
    export OPENAI_API_KEY=sk-***
    ```

    ```bash Windows
    setx OPENAI_API_KEY sk-***
    ```

    </CodeGroup>

  </Step>
  <Step title="Setup & start workspace">
    Setup the `ai-cookbook` workspace
    ```bash
    phi ws setup
    ```

    Start `ai-cookbook` resources
    ```bash
    phi ws up
    ```

  </Step>
  <Step title="Run a recipe">

    Run the `simple_conversation.py` recipe
    ```bash
    python recipe/simple_conversation.py
    ```

  </Step>
</Steps>
