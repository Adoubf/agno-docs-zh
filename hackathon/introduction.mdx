Here's a quick overview of the multi-modality support for each model:

- **Gemini** - Image, Audio and Video input
- **OpenAI** - Image and Audio input
- **Anthropic** - Image input
- **Groq** - Image input
- **Mistral** - Image input

Checkout the full [compatibility matrix](https://docs.agno.com/models/compatibility#multimodal-support) for more information.
