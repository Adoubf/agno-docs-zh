如果您运行 `ag auth` 时遇到错误：`CLI authentication failed` 或您的 CLI 卡在

```
Waiting for a response from browser...
```

这意味着您的 CLI 无法与 [app.agno.com](https://app.agno.com) 上的 Agno 账户完成认证。

最快的解决方法是导出您的 `AGNO_API_KEY` 环境变量。可以通过运行以下命令实现：

```bash
export AGNO_API_KEY=<your_api_key>
```

您的 API 密钥可在 [app.agno.com](https://app.agno.com/settings) 侧边栏的 `API Key` 中找到。

<img src="/images/cli-faq.png" alt="agno-api-key" width={600}/>

CLI 认证失败的原因：

- 部分浏览器（如 Safari 和 Brave）会阻止连接到本地主机域名。像 Chrome 这样的浏览器与 `ag setup` 配合良好。
