如果您在使用专有模型（如OpenAI模型）时遇到速率限制问题，我们提供了设置`exponential_backoff=True`和将`delay_between_retries`更改为以秒为单位的值（默认为1秒）的选项。

例如：  
```python
from agno.agent import Agent
from agno.models.openai import OpenAIChat

agent = Agent(
    model=OpenAIChat(id="gpt-4o"),
    description="You are an enthusiastic news reporter with a flair for storytelling!",
    markdown=True,
    exponential_backoff=True,
    delay_between_retries=2
)
agent.print_response("Tell me about a breaking news story from New York.", stream=True)
```

有关速率限制的具体信息，请参阅我们的[模型文档](../models/)。

对于OpenAI而言，他们采用分层速率限制机制。更多详情请查阅[官方文档](https://platform.openai.com/docs/guides/rate-limits/usage-tiers)。
