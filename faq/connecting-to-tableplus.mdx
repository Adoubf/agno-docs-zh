若需检查 pgvector 容器以探索存储或知识库，您可以使用 TablePlus。请按以下步骤操作：

## 步骤 1：启动 `pgvector` 容器

运行以下命令在本地启动 `pgvector` 容器：

```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agno/pgvector:16
```

- `POSTGRES_DB=ai` 设置默认数据库名称
- `POSTGRES_USER=ai` 和 `POSTGRES_PASSWORD=ai` 定义数据库凭证
- 该容器将暴露端口 `5432`（映射至本地机器的 `5532` 端口）

## 步骤 2：配置 TablePlus

1. **打开 TablePlus**：启动 TablePlus 应用程序
2. **新建连接**：点击 `+` 图标添加新连接
3. **选择 `PostgreSQL`**：选择 PostgreSQL 作为数据库类型

填写以下连接信息：

- **主机**：`localhost`
- **端口**：`5532`
- **数据库**：`ai`
- **用户**：`ai`
- **密码**：`ai`
