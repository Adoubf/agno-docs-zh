`Shell` enables an Assistant to interact with the shell.

## Functions

<ResponseField name="run_shell_command">
  Runs a shell command and returns the output or error.
</ResponseField>

## Example

The following assistant will run a shell command and show contents of the current directory. Note: Mention your OS to the assistant to make sure it runs the correct command.

```python cookbook/tools/shell.py
from phi.assistant import Assistant
from phi.tools.python import ShellTools

assistant = Assistant(tools=[ShellTools()], show_tool_calls=True)
assistant.print_response("Show me the contents of the current directory")
```
