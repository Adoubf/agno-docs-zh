**Phidata provides a powerful set of building blocks for your AI product:**

- [Assistants](/blocks/assistant/introduction) enable LLMs to achieve tasks using [function calling](https://platform.openai.com/docs/guides/function-calling).
- [Custom Assistants](/blocks/assistant/custom/introduction) are pre-built assistants designed for specific tasks.
- [Storage](/blocks/storage/introduction) provides long-term memory for assistants.
- [Knowledge Bases](/blocks/kb/introduction) provide assistants with information to improve their responses.
- [Vector Databases](/blocks/vectordb/introduction) provide storage for knowledge bases.
- [LLMs](/blocks/llm/introduction) are machine-learning models that have been trained to understand natural language and code. They power the assistants' decision making using [function calling](https://platform.openai.com/docs/guides/function-calling).
- [Apps](/blocks/app/introduction) are tools like FastApi, Streamlit, Django that serve Assistants as AI Apps.
- [Resources](/blocks/resource/introduction) are infrastructure resources that run AI Apps. Eg: Docker containers, ECS services
- [Workspace](/blocks/workspace/introduction) is the codebase for our application and helps manage our Apps and Resources in one place.
