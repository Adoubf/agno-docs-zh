---
title: Memory
---

Every `Assistant` comes with built-in memory that can be used to access the **Chat History**, **LLM Messages** or **References**. Access the assistant memory using `assistant.memory`

## Usage

```python assistant_memory.py
from phi.assistant import Assistant, AssistantMemory

assistant = Assistant()

# -*- Print a response
assistant.print_response('Share a 5 word horror story.')

# -*- Get the memory
memory: AssistantMemory = assistant.memory

# -*- Print Chat History
print(f"Chat History: {memory.chat_history}")

# -*- Print LLM Messages
print(f"LLM Messages: {memory.llm_messages}")
```

<Note>

AssistantMemory only lasts while the session is active. To provide memory across sessions, remember to add `Storage` to the `Assistant`.

</Note>

## Params

<ResponseField name="chat_history" type="List[Message]">
  Messages between the user and the LLM. The actual prompts are stored in the
  `llm_messages`.
</ResponseField>
<ResponseField name="llm_messages" type="List[Message]">
  Prompts sent to the LLM and the LLM responses.
</ResponseField>
<ResponseField name="references" type="List[References]">
  References from the vector database.
</ResponseField>

## Example

<Steps>
  <Step title="Create a virtual environment">
    Open the `Terminal` and create an `ai` directory with a python virtual environment.

    <CodeGroup>

    ```bash Mac
    mkdir ai && cd ai

    python3 -m venv aienv
    source aienv/bin/activate
    ```

    ```bash Windows
    mkdir ai; cd ai

    python3 -m venv aienv
    aienv/scripts/activate
    ```

    </CodeGroup>

  </Step>
  <Step title="Install libraries">
    Install `phidata` and `openai` using pip

    <CodeGroup>

    ```bash Mac
    pip install -U phidata openai
    ```

    ```bash Windows
    pip install -U phidata openai
    ```

    </CodeGroup>

  </Step>
  <Step title="Create an assistant">
    Create a file `assistant_memory.py` with the following contents

```python assistant_memory.py
from phi.assistant import Assistant, AssistantMemory

assistant = Assistant()

# -*- Print a response
assistant.print_response('Share a 5 word horror story.')

# -*- Get the memory
memory: AssistantMemory = assistant.memory

# -*- Print Chat History
print(f"Chat History: {memory.chat_history}")

# -*- Print LLM Messages
print(f"LLM Messages: {memory.llm_messages}")
```

  </Step>
  <Step title="Run the assistant">
    Run the `assistant_memory.py` file

```bash
python assistant_memory.py
```

  </Step>
</Steps>
