Agno 支持使用 PostgreSQL 作为智能体的存储后端，通过 `PostgresStorage` 类实现。

## 使用方法

### 运行 PgVector

安装 [docker desktop](https://docs.docker.com/desktop/install/mac-install/) 并通过以下命令在 **5532** 端口运行 **PgVector**：

```bash
docker run -d \
  -e POSTGRES_DB=ai \
  -e POSTGRES_USER=ai \
  -e POSTGRES_PASSWORD=ai \
  -e PGDATA=/var/lib/postgresql/data/pgdata \
  -v pgvolume:/var/lib/postgresql/data \
  -p 5532:5432 \
  --name pgvector \
  agno/pgvector:16
```

```python postgres_storage_for_agent.py
from agno.storage.postgres import PostgresStorage

db_url = "postgresql+psycopg://ai:ai@localhost:5532/ai"

# 使用PostgreSQL数据库创建存储后端
storage = PostgresStorage(
    # 将会话存储在 ai.sessions 表中
    table_name="agent_sessions",
    # db_url: PostgreSQL 数据库 URL
    db_url=db_url,
)

# 为智能体添加存储功能
agent = Agent(storage=storage)
```
