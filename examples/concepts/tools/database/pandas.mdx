描述：

使用agno库结合PandasTools实现了一个用于自动化数据分析的AI智能体。

该智能体能加载CSV文件（data.csv）并根据自然语言指令执行数据分析。

无需手动编写Pandas代码即可与数据交互，简化了数据探索和洞察提取流程。

包含环境变量和依赖项的设置说明。

---
标题: Pandas工具集
---

## 代码

```python cookbook/tools/pandas_tools.py
from agno.agent import Agent
from agno.tools.pandas import PandasTools

agent = Agent(
    tools=[PandasTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Load and analyze the dataset from data.csv")
```
