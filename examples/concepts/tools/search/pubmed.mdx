## 代码

```python cookbook/tools/pubmed_tools.py
from agno.agent import Agent
from agno.tools.pubmed import PubMedTools

agent = Agent(
    tools=[PubMedTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Find recent research papers about COVID-19 vaccines")
```
