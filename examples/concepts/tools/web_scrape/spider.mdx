## 代码

```python cookbook/tools/spider_tools.py
from agno.agent import Agent
from agno.tools.spider import SpiderTools

agent = Agent(
    tools=[SpiderTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Crawl https://example.com and extract all links")
```
