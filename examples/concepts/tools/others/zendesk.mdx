## 代码

```python cookbook/tools/zendesk_tools.py
from agno.agent import Agent
from agno.tools.zendesk import ZendeskTools

agent = Agent(
    tools=[ZendeskTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Show me all open tickets")
```
