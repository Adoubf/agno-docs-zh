## 代码

```python cookbook/tools/openbb_tools.py
from agno.agent import Agent
from agno.tools.openbb import OpenBBTools

agent = Agent(
    tools=[OpenBBTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Get the latest stock price for AAPL")
```
