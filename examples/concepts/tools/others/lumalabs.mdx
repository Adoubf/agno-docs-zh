## 代码

```python cookbook/tools/lumalabs_tools.py
from agno.agent import Agent
from agno.tools.lumalabs import LumaLabsTools

agent = Agent(
    tools=[LumaLabsTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Generate a 3D model of a futuristic city")
```
