## 代码

```python cookbook/tools/x_tools.py
from agno.agent import Agent
from agno.tools.x import XTools

agent = Agent(
    tools=[XTools()],
    show_tool_calls=True,
    markdown=True,
)
agent.print_response("Make a post saying 'Hello World from Agno!'")
```
