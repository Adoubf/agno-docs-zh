## 代码

```python cookbook/tools/whatsapp_tools.py
from agno.agent import Agent
from agno.models.google import Gemini
from agno.tools.whatsapp import WhatsAppTools

agent = Agent(
    name="whatsapp",
    model=Gemini(id="gemini-2.0-flash"),
    tools=[WhatsAppTools()],
)

# 示例：发送模板消息
# 注意：将 '''hello_world''' 替换为您实际的模板名称
agent.print_response(
    "Send a template message using the '''hello_world''' template in English to +91 1234567890"
)
```

## 使用指南

<Steps>
  <Snippet file="create-venv-step.mdx" />

  <Step title="设置 WhatsApp Business API">
    1. 访问 [Meta开发者平台](https://developers.facebook.com/docs/whatsapp/cloud-api/get-started)
    2. 创建Meta应用并配置WhatsApp Business API
    3. 获取您的电话号码ID和永久系统用户访问令牌
  </Step>

  <Step title="设置您的API密钥和标识符">
    ```bash
    export WHATSAPP_ACCESS_TOKEN=xxx
    export WHATSAPP_PHONE_NUMBER_ID=xxx
    export OPENAI_API_KEY=xxx # Or your preferred LLM API key
    ```
  </Step>

  <Step title="安装库">
    ```bash
    pip install -U agno openai google-generativeai # Add any other necessary WhatsApp SDKs
    ```
  </Step>

  <Step title="运行智能体">
    <CodeGroup>
    ```bash Mac
    python cookbook/tools/whatsapp_tools.py
    ```

    ```bash Windows
    python cookbook/tools/whatsapp_tools.py
    ```
    </CodeGroup>
  </Step>
</Steps>
