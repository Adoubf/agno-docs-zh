## Example

```python cookbook/llms/anyscale/agent.py
from phi.agent import Agent
from phi.llm.anyscale import Anyscale

agent = Agent(
    llm=Anyscale(model="mistralai/Mixtral-8x7B-Instruct-v0.1"),
    description="You help people with their health and fitness goals.",
)
agent.print_response("Share a 2 sentence quick and healthy breakfast recipe.", markdown=True)
```
