---
title: 智能体化RAG
---
本示例应用展示了如何构建一个先进的RAG（检索增强生成）系统，该系统结合知识库搜索与大型语言模型，为数据提供深度洞察。

## 该智能体能够：
- 处理并理解多种来源的文档（PDF、网站、文本文件）
- 使用向量嵌入构建可搜索的知识库
- 保持跨会话的对话上下文和记忆
- 为回答提供相关引用和来源
- 生成摘要并提取关键见解
- 回答后续问题和澄清请求

## 该智能体采用：
- 向量相似性搜索实现相关文档检索
- 对话记忆机制提供上下文响应
- 引用追踪功能标注信息来源
- 动态知识库更新机制

## 可尝试的示例查询：
- "这份文档的关键要点是什么？"
- "能否总结主要论点及支撑证据？"
- "有哪些重要统计数据和发现？"
- "这与[主题X]有何关联？"
- "该分析存在哪些局限性或空白？"
- "能否详细解释[概念X]？"
- "其他哪些来源支持或反驳这些主张？"

## 使用说明

<Steps>
  <Step title="克隆代码仓库">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="创建虚拟环境">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```
  </Step>

  <Step title="安装依赖项">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/agentic_rag/requirements.txt
    ```
  </Step>

  <Step title="运行 PgVector">
    首先安装[Docker Desktop](https://docs.docker.com/desktop/install/mac-install/)。

    然后通过辅助脚本运行：
    ```bash
    ./cookbook/scripts/run_pgvector.sh
    ```

    或直接使用Docker运行：
    ```bash
    docker run -d \
      -e POSTGRES_DB=ai \
      -e POSTGRES_USER=ai \
      -e POSTGRES_PASSWORD=ai \
      -e PGDATA=/var/lib/postgresql/data/pgdata \
      -v pgvolume:/var/lib/postgresql/data \
      -p 5532:5432 \
      --name pgvector \
      agnohq/pgvector:16
    ```
  </Step>


  <Step title="设置API密钥">
    ```bash
    # 必须
    export OPENAI_API_KEY=***
    # 可选
    export ANTHROPIC_API_KEY=***
    export GOOGLE_API_KEY=***

    ```
    推荐使用gpt-4o以获得最佳性能。
  </Step>

  <Step title="启动应用">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/agentic_rag/app.py
    ```
    访问[localhost:8501](http://localhost:8501)开始使用Agentic RAG。
  </Step>
</Steps>
