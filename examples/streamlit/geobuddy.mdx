GeoBuddy 是一个地理分析智能体，通过解析图像中的地标、建筑和文化符号等视觉线索来预测地理位置。

### 核心功能
- 地理位置识别：根据上传图像预测位置信息  
- 详细推理分析：基于视觉线索解释预测依据  
- 友好用户界面：采用 Streamlit 构建直观体验

### 简单使用示例  
- 自然景观：城市天际线、山脉全景或著名地标  
- 建筑特征：独特建筑物、桥梁或特色城市景观  
- 文化线索：招牌文字、语言提示、国旗或特色服饰  

### 高级用法  
尝试提供包含细微特征的图像，如不同语言的店铺招牌或非全球知名的标志性地标。GeoBuddy 将更深层次地分析建筑风格、环境特征（如沙漠 vs 热带）和文化参照物。

### 使用说明  
<Steps>  
  <Step title="克隆该仓库">  
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```  
  </Step>  

  <Step title="创建虚拟环境">  
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```  
  </Step>  

  <Step title="安装依赖">  
    ```bash
    pip install -r cookbook/examples/streamlit_apps/geobuddy/requirements.txt
    ```  
  </Step>  

  <Step title="设置 API 密钥">  
    GeoBuddy 使用 Google PaLM API 进行高级图像推理：  

    ```bash
    export GOOGLE_API_KEY=***
    ```  
  </Step>  

  <Step title="启动应用">  
    ```bash
    streamlit run cookbook/examples/streamlit_apps/geobuddy/app.py
    ```  
  </Step>  

  <Step title="打开应用">  
    在浏览器中访问 http://localhost:8501 即可使用 GeoBuddy。  
  </Step>  
</Steps>  

### 专业建议  
- 高分辨率图像：带有清晰招牌或地标的图像可提高准确率  
- 多角度拍摄：不同视角（如街景 vs 航拍）能展现独特线索  
- 上下文线索：车牌、本地建筑元素甚至植被等细节都可能影响定位结果
