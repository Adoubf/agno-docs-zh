title: 棋局对战
---
象棋对战是一个多智能体协作下棋的应用程序，展示了复杂游戏环境中多智能体系统的强大能力。

### 核心能力
- 多智能体系统：包含白棋智能体和黑棋智能体负责走子决策  
- 走子验证：专用合法走子智能体确保符合游戏规则  
- 游戏协调：主控智能体统筹游戏流程和结束条件  
- 交互式界面：使用 Streamlit 构建实时游戏可视化

### 系统组件
- 白棋智能体：制定策略并选择白棋走法  
- 黑棋智能体：控制并决定黑棋走法  
- 合法走子智能体：根据象棋规则验证所有走法  
- 主控智能体：协调游戏流程并监控游戏状态  

### 高级特性
系统展示了复杂的智能体交互，每个AI组件都有特定角色。智能体通过通信与协作构建完整的下棋体验，呈现了多个专业AI如何高效协同工作。

### 代码
完整代码参见 [Agno仓库](https://github.com/agno-agi/agno/tree/main/cookbook/examples/streamlit_apps/chess_team)。

### 使用指南
<Steps>
  <Step title="克隆该仓库">
    ```bash
    git clone https://github.com/agno-agi/agno.git
    cd agno
    ```
  </Step>

  <Step title="创建虚拟环境">
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
    ```
  </Step>

  <Step title="安装依赖">
    ```bash
    pip install -r cookbook/examples/streamlit_apps/chess_team/requirements.txt
    ```
  </Step>

  <Step title="设置 API 密钥">
    象棋团队智能体使用Anthropic API进行推理：

    ```bash
    export ANTHROPIC_API_KEY=your_api_key_here
    ```
  </Step>

  <Step title="启动应用">
    ```bash
    streamlit run cookbook/examples/streamlit_apps/chess_team/app.py
    ```
  </Step>

  <Step title="打开应用">
    然后在浏览器中打开 http://localhost:8501 即可观看AI智能体对弈。
  </Step>
</Steps>
