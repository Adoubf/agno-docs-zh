## SqlAgentStorage Params

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `table_name` | `str` | - | The name of the table to store agent conversations in. |
| `db_url` | `str` | - | The database URL to connect to. |
| `db_engine` | `Engine` | - | The database engine to use. |
| `metadata` | `MetaData` | - | The metadata to use for the database. |
