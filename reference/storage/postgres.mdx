## Example

```python agent_with_storage.py
from phi.agent import Agent
from phi.storage.agent.postgres import PgAgentStorage

agent = Agent(
    # -*- Add storage to the agent
    storage = PgAgentStorage(
        # Table name: ai.agent_runs
        table_name="agent_runs",
        db_url=db_url,
    )
)
```

## PgAgentStorage Params

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `table_name` | `str` | - | The name of the table to store agent conversations in. |
| `schema` | `str` | `"llm"` | The schema to store the table in. |
| `db_url` | `str` | - | The database URL to connect to. |
| `db_engine` | `Engine` | - | The database engine to use. |
| `metadata` | `MetaData` | - | The metadata to use. |
