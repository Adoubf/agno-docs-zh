## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.llm.openai import OpenAIChat

agent = Agent(
    llm=OpenAIChat(
        model="gpt-4",
        max_tokens=1024,
        temperature=0.9,
    )
)

# -*- Print a response
agent.print_response('Share a 5 word horror story.')
```

</CodeGroup>
