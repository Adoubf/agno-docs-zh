## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.llm.azure import AzureOpenAIChat

agent = Agent(
    llm=AzureOpenAIChat(model="gpt-35-turbo"),
    description="You help people with their health and fitness goals.",
)
agent.print_response("Share a quick healthy breakfast recipe.", markdown=True)
```

</CodeGroup>

## Azure Params

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `name` | `str` | `"AzureOpenAIChat"` | The name of the Azure OpenAI Chat model. |
| `model` | `str` | - | The specific model to use for Azure OpenAI Chat. |
| `api_key` | `str` | - | The API key for Azure OpenAI. |
| `api_version` | `str` | `"2023-12-01-preview"` | The API version to use for Azure OpenAI. |
| `azure_deployment` | `str` | - | The Azure deployment name for the model. |
| `base_url` | `str` | - | The base URL for the Azure OpenAI API. |
| `azure_ad_token` | `str` | - | The Azure AD token for authentication. |
| `azure_ad_token_provider` | `str` | - | The provider for Azure AD token. |
| `openai_client` | `AzureOpenAIClient` | - | An instance of AzureOpenAIClient, if already created. |
