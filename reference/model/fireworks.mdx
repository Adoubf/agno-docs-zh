## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.llm.fireworks import Fireworks

agent = Agent(
    llm=Fireworks(model="accounts/fireworks/models/firefunction-v1"),
    description="You help people with their health and fitness goals.",
)
agent.print_response("Share a quick healthy breakfast recipe.", markdown=True)
```
</CodeGroup>

## Fireworks Params

| Parameter | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| `name` | `str` | `"Fireworks"` | The name of the Fireworks LLM instance |
| `model` | `str` | `"accounts/fireworks/models/firefunction-v1"` | The Fireworks model to use |
| `api_key` | `str` |`None` | Your Fireworks API key |
| `base_url` | `str` | `"https://api.fireworks.ai/inference/v1"` | The base URL for Fireworks API |

## LLM Params

`Fireworks` is a subclass of the `LLM` class and has access to the same params
