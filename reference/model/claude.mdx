## Example

<CodeGroup>

```python agent.py
from phi.agent import Agent
from phi.llm.anthropic import Claude

agent = Agent(
    llm=Claude(model="claude-3-opus-20240229"),
    description="You help people with their health and fitness goals.",
)
agent.print_response("Share a quick healthy breakfast recipe.", markdown=True)
```

</CodeGroup>

## Claude Params

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | `str` | `"Claude"` | Name of the Claude model |
| `model` | `str` | `"claude-3-opus-20240229"` | The specific Claude model to use |
| `max_tokens` | `int` | `1024` | Maximum number of tokens to generate |
| `temperature` | `float` | - | Controls randomness in output generation |
| `stop_sequence` | `List[str]` | - | Sequences where the model should stop generating |
| `top_p` | `float` | - | Controls diversity of output |
| `top_k` | `int` | - | Limits vocabulary for next-word selection |
| `api_key` | `str` | - | API key for Anthropic |
| `base_url` | `str` | - | Base URL for the Anthropic API |
| `anthropic_client` | `AnthropicClient` | - | Custom Anthropic client, if provided |
