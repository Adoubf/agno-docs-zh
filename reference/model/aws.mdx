## AWS Bedrock Params

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | `str` | `"AwsBedrock"` | Name of the AWS Bedrock model |
| `model` | `str` | `"anthropic.claude-v2"` | The specific model to use from AWS Bedrock |
| `aws_region` | `str` | `None` | AWS region to use for the Bedrock service |
| `aws_profile` | `str` | `None` | AWS profile to use for authentication |
| `aws_client` | `AwsApiClient` | `None` | Custom AWS API client to use, if provided |
