The `Assistant` class provides an easy to use, human like interface to language models.

## Example

```python assistant.py
from phi.assistant import Assistant

assistant = Assistant()

# -*- Print a response
assistant.print_response('Share a quick healthy breakfast recipe.')

# -*- Get the response as a string
response = assistant.chat('Share a quick healthy breakfast recipe.', stream=False)

# -*- Get the response as a stream
response = ""
for delta in assistant.chat('Share a quick healthy breakfast recipe.'):
    response += delta
```
