## 代码格式化

采用统一标准格式化代码库能节省时间和心智负担。Agno模板预先配置了[ruff](https://docs.astral.sh/ruff/)，您可以通过辅助脚本或直接运行。

<CodeGroup>

```bash 终端
./scripts/format.sh
```

```bash ruff
ruff format .
```

</CodeGroup>

## 验证

代码静态检查和类型检查为代码库提供额外保护层。我们强烈建议在推送任何更改前运行验证脚本。

Agno模板预先配置了[ruff](https://docs.astral.sh/ruff/)和[mypy](https://mypy.readthedocs.io/en/stable/)，您可以通过辅助脚本或直接运行。具体配置请查看`pyproject.toml`文件。

<CodeGroup>

```bash 终端
./scripts/validate.sh
```

```bash ruff
ruff check .
```

```bash mypy
mypy .
```

</CodeGroup>
