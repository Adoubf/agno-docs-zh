---
title: 创建 Git 仓库
---
创建一个 git 仓库来与团队共享您的应用程序。

<Steps>
  <Step title="创建一个 Git 仓库">
    创建一个新的 [git 仓库](https://github.com/new)。
  </Step>
  <Step title="推送你的代码">
    将您的代码推送至该 git 仓库。

    ```bash 终端
    git init
    git add .
    git commit -m "Init LLM App"
    git branch -M main
    git remote add origin https://github.com/[YOUR_GIT_REPO].git
    git push -u origin main
    ```

  </Step>
  <Step title="邀请你的团队加入">
    请团队成员按照 [新用户设置步骤](/workspaces/workspace-management/new-users) 来使用此 workspace。
  </Step>

</Steps>
