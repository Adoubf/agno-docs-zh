在当前目录创建新工作区。

## 参数

<ResponseField name="name" type="str">
 新工作区名称。`--name` `-n`
</ResponseField>
<ResponseField name="template" type="str">
 工作区初始模板。`--template` `-t`
</ResponseField>
<ResponseField name="url" type="str">
 初始模板URL。`--url` `-u`
</ResponseField>
<ResponseField name="print_debug_log" type="bool">
 打印调试日志。`--debug` `-d`
</ResponseField>
