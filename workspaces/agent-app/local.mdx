Agent App 是我们构建智能体系统的核心工作区。它包含：

- 用于部署智能体、团队和工作流程的 FastAPI 服务器
- 用于调试和测试的 streamlit 应用。该 streamlit 应用功能多样，可作为智能体系统的管理界面并展示各类数据
- 用于会话和向量存储的 postgres 数据库

该系统设计为本地通过 docker 运行，生产环境部署于 AWS。
