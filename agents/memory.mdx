---
title: Memory
---

Every Agent comes with built-in memory that can be used to access the **Chat History** or **LLM Messages**. Access the agent memory using `agent.memory`
- **Chat History:** The messages between the user and LLM.
- **LLM Messages:** The full list of messages sent to the LLM, including system prompt, tool calls etc.
- **Personalized Memories:** Memories created by that agent personalized to the user.
- **Session Summary:** A summary of the conversation.

## Example

```python agent_memory.py
from rich.pretty import pprint

from phi.agent import Agent, AgentMemory
from phi.tools.yfinance import YFinanceTools

agent = Agent(tools=[YFinanceTools()])

# -*- Print a response
agent.print_response("What is the price of AAPL?", stream=True)

# -*- Get the memory
memory: AgentMemory = agent.memory

# -*- Print Chat History
print("============ Chat History ============")
pprint(memory.chat_history)

# -*- Print Run Messages
print("============ Run Messages ============")
pprint(memory.run_messages)
```

<Warning>

AgentMemory only lasts while the session is active. To provide memory across sessions, add `Storage` to the `Agent`.

</Warning>

## Params

<ResponseField name="memory" type="AgentMemory" default="AgentMemory()">
  Agent's memory object used for storing and retrieving information.
</ResponseField>
<ResponseField
  name="add_history_to_messages"
  type="bool"
  default="False"
  alias="add_chat_history_to_messages"
>
  If true, adds the chat history to the messages sent to the Model.
</ResponseField>
<ResponseField name="num_history_responses" type="int" default="3">
  Number of historical responses to add to the messages.
</ResponseField>
<ResponseField name="create_user_memories" type="bool" default="False">
  If true, create and store personalized memories for the user.
</ResponseField>
<ResponseField name="update_user_memories_after_run" type="bool" default="True">
  If true, update memories for the user after each run.
</ResponseField>
<ResponseField name="create_session_summary" type="bool" default="False">
  If true, create and store session summaries.
</ResponseField>
<ResponseField
  name="update_session_summary_after_run"
  type="bool"
  default="True"
>
  If true, update session summaries after each run.
</ResponseField>
