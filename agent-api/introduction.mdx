欢迎使用Simple Agent API：这是一个稳健的、可用于生产环境的应用程序，用于将智能体作为API提供服务。它包含：

- 一个用于处理API请求的FastAPI服务器
- 一个用于存储智能体会话、知识和记忆的PostgreSQL数据库
- 一组预构建的智能体作为开发起点
