工程师和研究人员使用Agno构建以下智能体：

- **第一级**：具备工具和指令的智能体（[示例](/introduction/agents#level-1%3A-agents-with-tools-and-instructions)）
- **第二级**：具备知识和存储的智能体（[示例](/introduction/agents#level-2%3A-agents-with-knowledge-and-storage)）
- **第三级**：具备记忆和推理能力的智能体（[示例](/introduction/agents#level-3%3A-agents-with-memory-and-reasoning)）
- **第四级**：可推理协作的智能体团队（[示例](/introduction/multi-agent-systems#level-4%3A-agent-teams-that-can-reason-and-collaborate)）
- **第五级**：具备状态和确定性的智能工作流（[示例](/introduction/multi-agent-systems#level-5%3A-agentic-workflows-with-state-and-determinism)）

**示例**：使用YFinance API解答问题的第一级推理智能体：

```python 金融推理智能体
from agno.agent import Agent
from agno.models.anthropic import Claude
from agno.tools.reasoning import ReasoningTools
from agno.tools.yfinance import YFinanceTools

reasoning_agent = Agent(
    model=Claude(id="claude-sonnet-4-20250514"),
    tools=[
        ReasoningTools(add_instructions=True),
        YFinanceTools(stock_price=True, analyst_recommendations=True, company_info=True, company_news=True),
    ],
    instructions="Use tables to display data.",
    markdown=True,
)
```

# 为什么选择Agno？

Agno助您构建一流的高性能智能体系统，节省大量研究和样板代码时间。其差异化特性包括：

- **模型无关**：统一接口支持23+模型供应商，无锁定
- **高性能**：智能体平均初始化仅需**~3微秒**，内存占用**~6.5Kib**
- **原生推理支持**：提供三种推理方式：推理模型、`ReasoningTools`或我们的定制方案`chain-of-thought`
- **原生多模态**：支持文本/图像/音频/视频输入输出
- **先进多智能体架构**：行业领先的智能体团队架构，具备推理、记忆和共享上下文
- **内置智能搜索**：运行时使用20+向量数据库搜索，提供全异步高性能Agentic RAG
- **内置记忆存储**：预置`Storage`和`Memory`驱动实现长期记忆和会话存储
- **结构化输出**：通过模型结构化输出或`json_mode`返回强类型响应
- **预置FastAPI路由**：分钟级生产部署
- **实时监控**：在[agno.com](https://app.agno.com)监控会话和性能
